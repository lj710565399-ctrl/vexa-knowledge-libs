# ACT-001 · 吴宇森 / 英雄本色（暴力浪漫）

## ① 一句话签名
双枪齐射、升格慢镜、白鸽掠过、教堂十字光、把杀戮拍成一场悲壮的芭蕾。

## ② 镜头语言颗粒
- **运镜**：横向滑轨 + 360°环绕对峙；子弹时刻常用甩镜接升格；双人背靠背对枪用缓推压迫
- **景别**：中近景双枪持械手部特写 ↔ 广角全景人物腾空；对峙必给“双方同框”的对称构图
- **打光**：硬侧光 + 烟雾丁达尔光束（教堂彩窗/百叶窗格栅光）；逆光剪影制造殉道感
- **调色**：暖金 + 血红 + 冷蓝夜，高对比；阴影压死、高光保留
- **节奏/剪辑**：静默对视拉长 → 第一枪瞬间起爆 → 升格慢镜把一秒拉成五秒，弹壳、玻璃、白鸽同框慢落；快慢交替（正常帧扫射 ↔ 50–120fps 升格落幅）
- **符号**：白鸽（灵魂/纯洁穿过暴力）、点燃的钞票/香烟、教堂十字、风衣翻飞

## ③ VEXA-09 可复用改编
VEXA-09 与对手在 Caster's Bar 石墙象形浮雕间双枪对峙：360°环绕滑轨锁双方同框 → 蓝色霓虹替代教堂彩窗打出格栅光束 → 第一发能量弹起爆瞬间切 60fps 升格，**机械白鸽 / 数据残片**代替白鸽掠过慢落，弹壳与晶核火花同框 → 逆光把 VEXA-09 紫色装甲压成殉道剪影。用在“被迫开战、为同伴挡枪”这类情义爆点最对味。
