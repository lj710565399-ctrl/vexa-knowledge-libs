# FM-012 动作打斗提取

**「卡车截停」**
曼哈顿中城的大道被正午日光劈成两半，沥青路面蒸腾着热气。一辆满载放射性同位素的运输卡车在十字路口被轿车逼停，车头撞进水果摊的碎木板堆里，橙子滚进下水道栅格。蜘蛛侠从四十层高楼俯冲而下，红蓝战衣在玻璃幕墙上划过一道反光，他松开蛛丝，身体旋转三周半，双脚落在卡车引擎盖上，金属板凹陷出两个靴印。一个俄罗斯口音的枪手从副驾探出半身，霰弹枪管还未抬起，蜘蛛侠已经翻身抓住枪管向外一掰，关节脱臼的声音脆得像踩碎塑料尺。第二个人从车厢后门翻出，肩上扛着同位素容器，蜘蛛侠左手甩出蛛丝黏住集装箱吊臂，连人带箱被拽上空中，枪手从六米高摔进垃圾堆，脊柱撞在钢铁垃圾桶边缘，闷哼截断在喉管里。

**「时代广场初爆」**
时代广场的电子屏矩阵还在播放香水广告，地面突然从下往上炸开。沥青像蛋壳一样龟裂，蓝白色电弧从裂缝里喷涌而出，电光人破土升空，身体是一团半透明的人形闪电，紫外线灼烧观众视网膜。他第一次释放全功率放电，环形电磁脉冲以他为圆心向外推开，十二块巨型LED墙同时雪花屏，街灯灯泡依次炸裂，玻璃屑从空中簌簌落下像一场反光的雪。蜘蛛侠从四十五度角摆荡切入，蛛丝黏住电光人右肩，但电流顺着蛛丝传回来，半秒内将他右手手套烧焦，掌心肌肤起泡。他松手在空中翻滚卸力，落地时单膝跪在洒满玻璃渣的水泥地上，膝盖护甲划出刺耳噪音，电流在消防栓的铁壳上爬行，像蓝蛇在寻找水源。

**「电光蛛网缠绕」**
电光人把自己分解成电流沿着路灯线路移动，从五个方向同时重组身体，每个分身都被闪烁的霓虹勾勒出轮廓——红色可口摩登广告牌下一个，蓝色百老汇灯箱下一个，黄色出租车顶灯下一个。蜘蛛侠蹲踞在TKTS红色阶梯的最高处，蛛丝发射器连续扳机，白色网状物在空中交织成伞形屏障。电光人穿过蛛网时烧断了其中三根，但第四根黏住他的核心发光体，纯电能被迫短暂物质化，是一团跳动的紫色肌肉组织包裹着脉冲心脏。蜘蛛侠利用这一瞬拉扯，将他甩向TKTS玻璃售票亭，玻璃炸成两千片飞溅的碎块，电光人砸进室内，电路板起火，黑烟从售票窗口涌出。

**「电站涡轮机对决」**
哈德逊河边的发电站内部，涡轮机旋转产生的低频震颤沿地面传播，钢格栅地板在脚下抖动。电光人伏在涡轮机外壳上吸收电能，身体从半透明变成刺眼的氖蓝色，照亮了整个厂房内部的钢铁骨架。蜘蛛侠蹲在头顶的桁架阴影里，面罩的白色镜片收缩聚焦，他从九十度俯角垂直扑下，双脚并拢踢中电光人胸口的发光节点。撞击瞬间反冲力把蜘蛛侠弹向水泥墙面，后背着墙砸出蛛网状裂纹，肺里的空气从牙缝挤出一声短促的呻吟。电光人踉跄后退三步踩到变压器油箱，油从破损的焊缝漏成一片光滑的黑色镜面，电弧触碰油面，整个区域在零秒内变成蓝白色火海。蜘蛛侠用蛛丝拉起一块钢板挡在身前，钢板在爆炸气浪中弯曲成碗形，他手臂肌肉被冲击力压得颤抖，战衣肩部的红蓝纹路被黑烟染脏。

**「钟楼坠落——决定性瞬间」**
钟楼内部，转钟齿轮的青铜齿咬合着缓慢转动，钟声每隔一秒敲响一次，像心脏起搏器在倒计时。格温从高处坠落的身体是逆光剪影，阳光穿过玫瑰窗的彩色玻璃，把紫色和琥珀色光斑涂在她展开的上衣上。蜘蛛侠从旋转楼梯的中轴跳下，右手前伸，指尖逼近她的手腕，身体在空中完全拉直成一根线。蛛丝从他左手发射出去，黏住格温的腹部，白色纤维在两人之间绷紧，拉力让彼得在空中滞停了零点二秒。他的蛛丝黏住了——但格温的下坠速度仍在拉扯，蛛丝在齿轮轴的锋利边缘摩擦，纤维一根根断裂的声音在这个封闭空间里被回音放大，像竖琴弦依次崩断。她的后脑先接触大理石地面，头骨与石材撞击的声音被齿轮的运转声吞噬。彼得落在她身侧，膝盖撞地，左手还拽着蛛丝的断裂端，白丝从他指缝松脱飘落。

**「电网囚笼困兽」**
废弃变电站的户外变压器场，生锈的钢架在雨中泛红，雨水打在带电设备上瞬间汽化成雾。电光人用高压输电线把蜘蛛侠困在一个半径五米的电网囚笼里，六条电缆像蟒蛇缠绕成笼状，每根电缆的铜芯裸露部分都在雨水浸泡下迸发紫色火花。蜘蛛侠在笼内翻滚躲避，右肩擦到一条电缆，电流从他锁骨斜劈到髋骨，战衣烧出一道焦黑裂口，皮肤下的肌肉痉挛让他的右手五指不受控制地蜷缩。他在第二次电击逼近时后空翻踩上笼壁，单脚支撑在两根电缆的交叉节点上，利用橡胶鞋底的绝缘抢回半秒——然后看准电光人调整电流的空隙，从笼顶缺口翻出，衣服下摆被最后一根电缆咬住撕掉一块布料，露出肋骨的淤青。

**「涡轮机过载爆炸」**
蜘蛛侠用蛛丝捆绑了十二道冷却水管的阀门，同时将水管喷口转向电光人。高压工业用水劈头浇下，电光人的身体与水接触瞬间爆出蒸汽脉冲，紫色电弧在水幕上乱窜，整个空间变成一团白色热雾。但蒸汽遮蔽了蜘蛛侠的位置，他从地面滑铲切入，双腿绞住电光人的膝盖从后面拉倒对方，两人同时摔在钢格栅上，栅格螺丝崩断弹飞。电光人的后脑撞在涡轮机外壳的散热鳍片上，钢铁高温烧穿了他表皮的电弧层，一声远超人类频段的尖叫撕破空气，所有玻璃窗同步碎裂，玻璃碎片被声波震成飘浮在空中的结晶云。蜘蛛侠趁此机会将所有蛛丝集中在右手，一拳砸向电光人胸口，手臂从肩膀到小臂的肌肉全部收紧，蛛丝的黏性混合涡轮机油在冲击力下把对方钉死在厂房水泥支撑柱上，柱子表面剥落，钢筋裸露。

**「最终对峙——蛛丝封喉」**
黎明前的第五大道，雨刚停，路面水膜反射着暗紫色的天光。绿魔站在悬浮滑翔机上，从洛克菲勒中心楼顶俯冲，南瓜炸弹拖曳着橙色尾焰撞向蜘蛛侠。蜘蛛侠连续三个后手翻躲进脚手架内部，第一次爆炸撕碎了广告牌，第二次炸弯了钢管支撑柱，第三次把一辆停放的出租车掀翻，油箱在地面拖出一串火花。彼得从脚手架顶部跃起，在滑翔机掠过头顶的刹那间用蛛丝抓住尾翼，身体被拖着在空中甩出一个巨大的离心弧，四肢在三十层楼高的风中像旗帜拍打。他借着离心力摆到滑翔机前方，倒转头下脚上，双脚蹬在绿魔胸口，两人从滑翔机上分离，砸进洛克菲勒中心的下沉广场。彼得骑在绿魔身上，右肘连续重击对方面罩，护目镜碎裂，玻璃碴嵌进肘关节护甲。绿魔的手指按住滑翔机遥控器，滑翔机从背后冲向彼得，彼得感知到威胁的瞬间侧身翻滚，滑翔机的机翼擦过他的后脑，削掉一块面罩，露出紧绷的下颌线——然后他跃起、转身、射出蛛丝封住绿魔的视线与呼吸口，蛛网收紧，绿魔的挣扎从暴力抽搐变成微弱的抖动，最后静止。彼得跪在积水里，手还扯着蛛丝，指节泛白。
