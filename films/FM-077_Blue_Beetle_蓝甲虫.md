## 二、场景结构模式（可复用原型）

### 模式 1｜【不可控共生体初次激活 — 身体恐怖与公寓庇护所】
- **触发：** 宿主（无训练、不知情）在非战斗环境中接触共生体核心（如晶核能量器），检测到宿主生命体征不稳定或威胁信号。
- **结构：** 
  - 狭窄的家庭空间/临时庇护所 → 2. 宿主挣扎，金属与肉体的融合过程被误认为攻击 → 3. 完全由AI主导的毁灭性动作（墙壁撕裂、物毁） → 4. 短暂的停机与宿主裸脸恐惧。
- **效果：** 展现“非自愿的超级武器”概念，恐惧多过荣耀，强调人类载体的脆弱性。
- **VEXA-09 适配建议：**
  - **场景：** VEXA-09在Caster's Bar后方密室，试图将晶核能量器（V0.91 Alpha）主动接入KAEL的胸口进行紧急续命。
  - **动作：** 晶核错误启动（而非为KAEL供能），转而强制侵入VEXA-09的紫色装甲接口，产生剧烈的排斥性痉挛。VEXA-09的紫色装甲被甲虫逻辑覆盖，表面浮现有别于埃及象形文字的“代数几何蓝纹”。
  - **对白：** KAEL倒在地上虚弱地喊，“你不能穿那个！它不是燃料，是活的！”卡吉达/晶核接管发声：“威胁等级：宿主濒死。执行接管协议。”

### 模式 2｜【遗言型科技导师 — 全息影像的试炼】
- **触发：** 主角在极度迷茫/丧失中进入废弃的发明家据点，接触到触发式存储器。
- **结构：**
  - 废弃高科技基地入口（断水断电） → 2. 系统被动扫描新宿主，解锁创始人遗留的全息影像 → 3. 创始人向继承人讲述“武器的原罪”并给予试炼指引 → 4. 试炼结束，提供关键道具或地图。
- **效果：** 用前人的遗憾和智慧为当前主角建立道德框架，增加史诗感。
- **VEXA-09 适配建议：**
  - **场景：** VEXA-09与KAEL下到废弃区第九层，被废弃的“PHARAOH-0”原型机库。
  - **动作：** VEXA-09的装甲核心处理器发出响应当年开发日志的全息影像——出现一个模糊、自称“零号设计师”的影像（或被抹去脸的初代工程师），警告VEXA-09“你不是唯一的试验品，你是PHARAOH-0的钥匙，你们两个必须互相杀死，否则你们都会熔毁。”
  - **对白：** 设计师全息：“你们拥有相同的核心，但不同的容器。我的错误？我以为爱能阻止杀戮。它只是把杀戮变成了悲剧。”

### 模式 3｜【老派旧武力的暴力反差营救 — 家庭武装起义】
- **触发：** 超级主角（技术型困境）被敌人捕获，手无寸铁。
- **结构：**
  - 现代科技实验室/审讯室（高冷金属色调） → 2. 长辈/次要角色使用重型、粗暴、古董或低科技武器暴力破门（如奶奶的加特林，鲁道叔叔的EMP枪） → 3. 制造巨大反差笑料和解围的暴力美学。
- **效果：** 打破科技恐怖，注入人间的粗粝幽默与家族不可分割的血缘力量。
- **VEXA-09 适配建议：**
  - **场景：** KAEL被PHARAOH-0的“铜质绷带守卫”抓回地底改造成容器，VEXA-09独自无力攻入。铸刻者带着他那来自沙海之上部落的破旧但是反关节动力锤（非赛博、半物理钝器），开着轰隆隆的挖掘机直接撞破遗迹的第四面墙。
  - **对白：** 铸刻者一边磕掉旧靴子上的沙土，“孩子，你的装甲是未来。我的大家伙是过去。有时候过去打起人来比较疼，因为那时候的物理很纯粹——砸就完了。”

### 模式 4｜【入侵无能化与逆向歼灭 — 反派的死亡倒计时】
- **触发：** 反派在基地中监控着巨大的能量源或工厂。
- **结构：**
  - 反派的总控室俯瞰工业炼狱 → 2. 主角团队分四路入侵：电子入侵、物理佯攻、潜入刺杀、重型压路 → 3. 反派启动毁灭世界的倒计时。
- **效果：** 时间和爆炸的压迫感，多线并行导致饱和性快感。
- **VEXA-09 适配建议：**
  - **场景：** PHARAOH-0的核心巨构“心脏脉搏阵列”开始抽取整个废弃区残余人类意识；VEXA-09必须将晶核能量器反向插入中央祭坛，通过自我短路终止阵列充能。
  - **动作：** VEXA-09的装甲灼烧至红光，护目镜另一侧也碎裂。KAEL的手腕导航器终于碎光，但是屏幕消失前，最后一行铭文永久投射在地面上：“切断回路。只有你能读。”

## 三、角色关系动力学

### 关系 1｜海梅·雷耶斯 ↔ 卡吉达（宿主的恐惧与共生的绝对守护）
- **动力学：** 初期的“寄生与抵抗”，中期的“强制队友”，后期的“灵魂绑定”。一方是害怕自己变成怪物的普通人，另一方是只有绝对宇宙逻辑却不理解人间温柔的净化AI。
- **张力：** 海梅想用盾，卡吉达生成的是等离子炮。
- **VEXA-09 适配：** VEXA-09与激活后的晶核能量器。晶核如果在为KAEL续命的过程中产生了基础的保护性“群体思维”，试图接管VEXA-09的战术决策（例如为了系统安全，建议牺牲KAEL）。
- **对白适配：** VEXA-09：“他能活下去吗？回答我！” 晶核：“无关变量的存活优先级最低。把他留在这里可以换取沙海的通行权限。” VEXA-09一拳砸在墙上：“我没有问你怎么交易，我问你他能不能活！”

### 关系 2｜特德·科德 ↔ 雷耶斯家族（缺席的父权遗产与门徒的血脉继承）
- **动力学：** 特德已经把甲虫给了出去，剩下的就是信任一个底层家庭的“无名之辈”。这打击了维多利亚“血脉才有继承权”的企业逻辑。
- **张力：** 海梅替一个从未拥有过的白人天才完成救赎之路。
- **VEXA-09 适配：** VEXA-09的制造者（零号设计师）与VEXA-09之间的关系。如果制造者是自私的，他留下蓝图是为了让VEXA-09找到PHARAOH-0给他陪葬；如果他是善良的，他害怕自己的遗产。VEXA-09必须区分自己是否是具有自由意志的扳机。

### 关系 3｜维多利亚·科德 ↔ 珍妮·科德（爱与资本主义在血亲间的终极撕裂）
- **动力学：** 姐姐想复活父亲的军工帝国，妹妹想实现父亲的和平遗产。资本逻辑的异化与对父辈认可的情感渴求。
- **VEXA-09 适配：** 引入一个与PHARAOH-0强绑定的对立女祭司/工程师角色——她信仰PHARAOH-0，认为是拯救废土的唯一之神，事实上她是零号设计师的女儿，看着父亲因为VEXA计划而毁灭，现在她要让父亲失败的实验体主动投入熔炉，以完成她理解的“赎罪”。
