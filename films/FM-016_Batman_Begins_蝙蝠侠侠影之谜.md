## 二、场景结构模式（可复用原型）

### 模式 1：「创伤记忆 → 技术转化」

- **触发：** 主角经历某个无法用常理解释的创伤事件（掉入井中/父母被杀），该事件成为心理黑箱
- **结构：** 创伤记忆 → 逃避/流浪(重复) → 遇到外部引导者（杜卡德/福克斯）→ 将创伤物化为工具（蝙蝠战衣/蝙蝠车/装备）
- **效果：** 主角从受害者转为控制者，将恐惧从被动体验转变为主动操控手段
- **VEXA-09 适配：** VEXA-09 的"记忆 Glitch"（身份类触发）可参照此模式。建议在 **Caster's Bar** 场景之后，VEXA-09 因接触晶核能量器而产生第一次"闪回"，闪回内容不是完整的记忆，而是 **PHARAOH-0 的轮廓 / 某个废弃区深处的构造图**。KAEL 和铸刻者可在场，用"你这不叫回忆，叫硬件自检"来定义。

### 模式 2：「义警的双重面具构建」

- **触发：** 主角需要一个社会身份 (Bruce Wayne) 和一个行动身份 (Batman) 来分别执行"融入"和"摧毁"的功能
- **结构：** 真实身份被质疑/弱点（韦恩的软肋是瑞秋）→ 公开场合扮演废物/声名狼藉（Wayne 装傻）→ 暗处执行计划 → 身份危机（瑞秋认出布鲁斯是蝙蝠侠）
- **效果：** 产生叙事张力——观众知道他是谁，但剧中人不知道；身份成为保护机制也是囚笼
- **VEXA-09 适配：** VEXA-09 不需要双重身份，但她 **在 KAEL 面前塑造了一个"冷漠赏金猎人"的形象，却在战斗中展现出保护性本能**。借鉴此模式，可以在 **霓虹街区逃亡** 场景中，让 VEXA-09 在某一刻被另一个赏金猎人叫出"真名"（代码/编号，如 "NX-09"），而 KAEL 首次意识到她的身份并非全部公开。

### 模式 3：「导师的背叛即世界观的崩塌」

- **触发：** 主角信任的导师被揭示为最终反派（杜卡德 = 拉斯·艾尔·古尔）
- **结构：** 建立导师权威（杜卡德传授一切给布鲁斯）→ 确认盟友关系（布鲁斯拒绝杀犯人） → 导师"死亡"假象 → 中段反转揭示导师未死且是幕后黑手 → 最终物理与理念的双重对决
- **效果：** 使正反派的冲突从"打怪"升级为"信仰之战"，让观众对主角的成长路径产生怀疑
- **VEXA-09 适配：** **铸刻者**的角色定位最适合运用此模式。当前铸刻者帮助 VEXA-09 和 KAEL 获取晶核能量器，但他的身份和动机不明。可在后续场景中揭示：铸刻者曾是 **PHARAOH-0 的次级系统指挥官**，因某些原因被流放至废弃区第七层。他帮助 VEXA-09 并非出于善意，而是为了获取晶核能量器来修复 PHARAOH-0 的核心节点（终极背叛）。VEXA-09 将面临"帮助=自毁"的哲学抉择。

### 模式 4：「恐惧是武器，也是开关」

- **触发：** 反派使用恐惧作为社会控制手段（稻草人的恐惧毒素 + 拉斯·艾尔·古尔的群体毁灭）
- **结构：** 展示恐惧的破坏力（毒素致狂）→ 主角被恐惧击中并失去控制 → 自我重新定义恐惧（"我是恐惧的化身"）→ 利用恐惧反击
- **效果：** 将恐惧从弱点翻转为核心武器链
- **VEXA-09 适配：** KAEL 现在有 **72小时倒计时（剩余约11小时）**。恐惧的来源是"时间不足"。此模式可直接映射：VEXA-09 的恐惧不是死亡，而是"无法完成保护任务"。可以在**激活晶核能量器**的场景中，KAEL 因疼痛拒绝继续，VEXA-09 第一次情绪失控/声调升高，喊出类似"你不活下来，我做这一切就没有意义"的台词——恐惧转化为行动驱动力。
