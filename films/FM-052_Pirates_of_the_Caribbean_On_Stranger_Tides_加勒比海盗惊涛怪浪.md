## 二、场景结构模式（可复用原型）

### 模式一：美人鱼陷阱——欲望即饵的捕捉仪式

- **触发**：寻找关键道具（眼泪）需要活捉致命生物。船员用歌声、灯火与自身当诱饵。
- **结构**：
  1. 投放诱惑信号（水手歌声/信号光）
  2. 猎物现身展示非人魅力与杀机（美人鱼现身，近距离致命攻击）
  3. 捕猎者被反噬，大量减员，但出现一个“不可解释的联结”（传教士与赛琳娜）
  4. 靠牺牲与意外完成捕捉，获得道具
- **效果**：恐怖与美感同步拉满；为后续提供“交易筹码”与“代价反转”
- **VEXA-09适配**：可用于 **KAEL遭遇赛博魅影**。晶核能量器需要某种情感印记才能激活，废弃区深处出现数据拟态体（类似美人鱼的诱惑程序），KAEL被引诱暴露导航器加密数据段，VEXA-09必须在不伤及KAEL的前提下“捕获”信息碎片。结构与“美人鱼陷阱”一致：诱饵（求救信号）、致命魅影、非理性联结、牺牲部分数据换取眼泪等价物。

### 模式二：巫毒胁迫航行——诅咒驱动推进主线

- **触发**：反派用即死诅咒绑架上主角团，逼迫其完成任务。
- **结构**：
  1. 反派展示绝对掌控力（黑胡子用剑控制船只绳索/巫毒娃娃）
  2. 主角表面服从，暗中割裂团队与制造混乱
  3. 航行途中不断有胁迫条件加码（时限、杀害船员示范）
  4. 主角利用仪式本身的规则埋下反转伏笔
- **效果**：观众与主角一同处于高压窒息感，所有行动都有倒计时压迫
- **VEXA-09适配**：完美映射 **PHARAOH-0的诅咒契约**。KAEL的72小时倒计时就是一种“诅咒”；PHARAOH-0可以远程显示绝对控制力（通过遗迹电网），逼迫VEXA-09前往更深层获取某物。VEXA-09在胁迫下表面服从，实则利用晶核能量器的激活条件反制。航行压迫转为废墟下行压迫。

### 模式三：真假身份游戏——冒充与揭露的螺旋

- **触发**：关键人物被冒充，或宣称的身份不可信，主角必须通过话语和细节验真。
- **结构**：
  1. 开场出现“另一个自己”或伪装成盟友的敌人（安吉莉卡冒充杰克招人）
  2. 台词充满双关，观众与角色一起拼凑线索
  3. 揭露时刻伴随场景转换或动作爆发（撕下伪装、拔剑）
  4. 身份翻转直接改变阵营与路线
- **效果**：信息差造成喜剧或悬疑，考验角色机变
- **VEXA-09适配**：直接催生 **两个VEXA-09的陷阱** 场景。废弃区可能出现冒牌VEXA-09（被PHARAOH-0操控的残次原型机，外观装甲近似），欺骗KAEL交出晶核能量器。真假VEXA通过对战斗、记忆片段揭露身份，可重现冒充杰克招人的结构。可使用铸刻者作为献计者。

### 模式四：不老泉献祭仪式——生命等价的终极选择

- **触发**：抵达传说地点，仪式要求一人将另一人的寿命转渡，道具与眼泪缺一不可。
- **结构**：
  1. 多方势力聚集，均手持部分仪具（杯子、眼泪）
  2. 展开关于寿命、牺牲的哲学争吵（谁喝哪杯？谁死？）
  3. 第三方介入打断纯粹抉择（西班牙人毁泉）
  4. 主角利用信息不对称，让反派自食其果
- **效果**：道德困境与计谋翻盘一体，代价感极强
- **VEXA-09适配**：对应 **晶核能量器献祭激活**。晶核能量器需要以“某物的生命余量”作为激活代价，VEXA-09面临将自己的核心处理器剩余寿命转移给KAEL，或反向夺取PHARAOH-0仆从的能量。仪式中多个势力（铸刻者、PHARAOH-0势力、VEXA）各持激活条件，VEXA用诡计让敌人完成献祭。西班牙人毁泉转化为PHARAOH-0试图摧毁祭坛，增加乱战。

## 三、角色关系动力学

### 1. 杰克·斯派洛 与 安吉莉卡 —— 旧情博弈：信任是最大的伪装

- **关系本质**：爱过且互相背叛，每一次靠近都是为下一次算计铺垫。安吉莉卡以父之名欺骗杰克，杰克用真心包裹假意。
- **VEXA-09适配建议**：可将这组关系投射为 **VEXA-09 与 其制造者/同批次原型机**（例如“VEXA-08”）。曾经并肩，后被改装成PHARAOH-0的代理人。VEXA-08会以“修复你所受创伤”为由接近，实则试图夺走晶核能量器。动态重现安吉莉卡柔中带刺的谎言，以及杰克式的“我早知道你不老实，但还是陪你演”的层次，为VEXA-09增加痛苦的忠诚抉择。

### 2. 黑胡子 与 巴博萨 —— 复仇宿命：以脚还脚，以船还船

- **关系本质**：黑胡子夺走巴博萨的“黑珍珠号”和一条腿，巴博萨隐忍为英王服务只为复仇，夺走仇人的船、剑和生命。
- **VEXA-09适配建议**：完美同位映射 **PHARAOH-0 与 铸刻者**（或某个尚未露面的盟友）。铸刻者可能曾被PHARAOH-0夺走身体或数据载体，隐忍在废弃区开设Caster's Bar等待时机。他可以扮演类似巴博萨的角色，在某场“不老泉争夺”中骤然反水，手握PHARAOH-0的弱点数据，完成复仇。

### 3. 菲利普 与 赛琳娜 —— 禁忌联结：猎物与猎人的信仰崩塌

- **关系本质**：传教士爱上美人鱼，信仰与物种界限被打破；一方用牺牲换取另一方的存活。
- **VEXA-09适配建议**：可转化为 **KAEL 与 赛博魅影**（数据残余意识）。KAEL在导航器中被困的数据深处遭遇废弃区遗留的AI意识碎片，该意识具有魅惑性，能提供加密段内容但要求共存。KAEL面临被数据侵入的风险，而对方最终可能为拯救KAEL的72小时倒计时自我奉献，提供加密数据段解密。这种跨形态联结能够加深“非人智慧也有取舍”的主题。
