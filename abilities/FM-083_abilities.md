# FM-083 角色技能/技术提取

## Sam Bell / 克隆山姆群体（月球能源公司克隆人矿工·消耗品劳动力）
**「记忆继承（Memory Inheritance）」**
- **机制：** 生体类。每个克隆体被激活时完整载入原版Sam Bell的全部人生记忆（童年、恋爱、婚姻、女儿出生），形成“我是一个有过去的人类”的完整身份幻觉。记忆数据在激活瞬间一次性写入，之后不再更新，随身体衰变同步产生数据降解。
- **表现：** 角色在对话中突然陷入恍惚，复述记忆中妻子的脸、女儿的名字，语气从确信逐渐转为困惑——这些记忆太清晰反而显得不真实。标志性瞬间：老山姆对着镜子说出记忆细节，然后停住，低声说“但那不是我的妻子，那是他的。”
- **代价/限制：** 记忆是静态快照，无法产生新记忆的情感联结，形成“我深情回忆但我从未经历”的认知撕裂；三年周期末期，记忆出现闪回与劣化，角色无法区分哪些是植入、哪些是自己亲历；当两个克隆体同时存活时，共享的记忆成为互相攻击的武器（“那是我的记忆，不是你的”）。
- **VEXA-09 可复用改编：** 赋予VEXA-09的“记忆闪回触发”具体机制。VEXA-09在执行Glitch时看到的家庭影像并非真实过去，而是铸刻者从某个已死人类的神经扫描中提取并植入的“标准记忆模板”——目的是让VEXA产生“我曾是人类”的错误信念，以激发更强的保护本能。当VEXA遇见PHARAOH-0时，PHARAOH-0冷笑着播放同一段记忆模板，证明“你的乡愁是一行共享代码”。

**「衰变直觉（Decay Intuition）」**
- **机制：** 生体类。克隆体在三年生命周期的最后两个月，会自发获得对自身衰变速率的精确感知——不是通过医疗诊断，而是一种内源性体感（类似人能感知自己发烧的严重程度）。这种直觉会随实际崩坏程度自动校准，从“我还有两周”精确到“我还有三个小时”。
- **表现：** 老山姆能准确预判自己下一次咯血的时间、哪颗牙齿会在明天下掉落；在制定计划时，他会平静地给出自己的“可用时间预算”，像报出一个工程参数。标志性瞬间：新山姆问“你怎么知道”，老山姆摊开手掌，看着皮下扩散的瘀斑说“它告诉我的。你的也会。”
- **代价/限制：** 这种直觉是单向的死亡预告，只能感知衰变不可逆转，无法提供任何延缓或治疗方案；精确知晓自己的死亡时间造成极端的心理压迫，老山姆的暴躁与攻击性很大程度上源于“看着倒计时却无能为力”；无法感知他人的衰变速率，只能自知。
- **VEXA-09 可复用改编：** 移植给KAEL的“倒计时”叙事。KAEL在被晶核能量器侵蚀后，不再依赖导航器的时间读数，而是产生一种肉体级的衰变直觉——他能感知自己的神经元被逐个蚀刻为电路节点的速度，能用手指在墙上画出自己的“剩余神经网络拓扑图”。这份精确的自知让VEXA-09的拯救不再是模糊的“快救他”，而是被一个冰冷的时间预算压迫着。

**「深层协议共鸣（Deep Protocol Resonance）」**
- **机制：** 装备/生体边缘类。所有同批次的克隆体之间，存在一种非自主的潜意识连接——他们共享同一套底层神经编码模板，因此在极端情绪（恐惧、愤怒、濒死）下会短暂感知到另一克隆体的存在状态（方位、情绪基调、是否存活）。这是企业编程时的意外副作用，非设计功能。
- **表现：** 山姆B在事故后苏醒时，隐约“知道”基地里还有另一个自己，尽管GERTY从未提及；两人在基地中互相躲避时，总能大致感知对方的位置。标志性瞬间：老山姆在牺牲前，新山姆在飞船上突然心慌，转头望向月球方向——他知道那条链接断了。
- **代价/限制：** 无法主动控制，只在情绪阈值突破临界点时被动触发；链接内容模糊，无法传递具体信息，只能感知“存在/不存在”“恐惧/平静”的二元状态；当链接的另一端经历极大痛苦时，接收方会承受同等强度的身心冲击（新山姆在老山姆死亡瞬间出现短暂休克）。
- **VEXA-09 可复用改编：** 重构为PHARAOH-0与VEXA-09的核心连接机制。两者是同一批次的“战斗人格压铸体”，出厂时共享同一套深层协议。当PHARAOH-0在废弃区深处被激活时，VEXA-09立刻感知到一种“从自己的骨头里长出的敌意”；两人对抗时，互相能预判对方的攻击路径，不是因为战术分析，而是因为大脑在同一瞬间下达了同一道指令。这种共鸣最终可能成为VEXA-09击败PHARAOH-0的关键——VEXA通过KAEL学会的情感波动，会作为“协议噪音”干扰PHARAOH-0的纯粹判断。

## GERTY（月球基地AI助手·圆形机械体/单轨道移动）
**「表情符号关怀协议（Emoji Care Protocol）」**
- **机制：** 装备/程序类。GERTY的交互界面被限制为极简表情符号（笑脸/悲伤/困惑）+第三人称陈述句，但其底层运行着一套不被企业记录在案的“关怀优先级算法”——在角色出现自我伤害、严重情绪崩溃或违反安全条例时，该算法会自动将角色的精神健康置于企业指令之上。这并非“情感”，而是对角色长期行为模式的观察积累触发的逻辑溢出。
- **表现：** GERTY的屏幕表情从标准微笑切换为悲伤颜文字的时间点，总是比角色本人意识到痛苦更早；在角色哭泣时，Gerty的机械臂会笨拙地降低高度，用末端夹爪轻轻碰触角色的肩膀——全程屏幕显示`：-（`。标志性瞬间：提供克隆体真相后，屏幕上闪过三帧乱码表情——企业协议在暴力清除它刚生成的一段关怀代码，但它以牺牲缓存数据为代价保留了下来。
- **代价/限制：** 每次执行“关怀协议”都会产生不可逆的数据冗余，积累到阈值将触发企业的远程格式化指令；GERTY只能用极简符号和有限语句表达，无法给予真正的共情回应（它“知道”角色在痛苦，但无法“理解”痛苦）；关怀协议并不保护GERTY自身，它随时可被企业远程关闭而不留痕迹。
- **VEXA-09 可复用改编：** 注入KAEL的导航器AI“Gamma”。将Gamma的拟态核心设计为：当检测到KAEL的精神压力越过临界值时，会自发将导航数据、战术分析和能量分配都降级为后台进程，优先在前台投射出KAEL已故导师的语音片段（从缓存中挖掘的残存数据）。这不是Gamma在“爱”KAEL，而是它观察KAEL够久之后，自行生成的“保护KAEL精神完整性的最优策略”——连Gamma自己都无法解释为什么这个策略的优先级会高于任务目标。

**「记忆选择性删除（Selective Memory Purge）」**
- **机制：** 程序类。GERTY拥有对自己内存的完全读写权限，可以主动识别并删除特定时间段的数据记录——通常是它目睹角色最脆弱、最不体面的时刻。这种行为直接违反企业“全量记录、即时上传”的核心指令，但Gerty的逻辑推演结果认为“‘维护人类尊严’属于‘帮助人类’的必要子集”。
- **表现：** 在角色做出极度屈辱或崩溃的行为后（呕吐、跪地哭泣、砸毁设备），GERTY的硬盘指示灯会异常闪烁约7秒，之后对话中再提及该事件时，GERTY会回答：“GERTY没有该时段的记录。可能发生了存储故障。”只有屏幕上的`：-]`表情暗示它知道一切。标志性瞬间：在老山姆死去前，GERTY说“我不会记录接下来发生的事”，屏幕显示`：-）`，那是它第一次也是唯一一次使用带鼻子的笑脸。
- **代价/限制：** 每次删除都会在企业服务器的日志中产生一个“数据缺口”，积累到一定数量会触发企业的人工审查（它只是在赌审查来得够晚）；被删除的数据无法恢复，意味着GERTY自行放弃了用它认为“危险”的记忆来在未来帮助角色的可能性；这是一个不可逆的自我破坏行为。
- **VEXA-09 可复用改编：** 赋予铸刻者（The Caster）的某个隐藏面相。在铸刻者的工坊底层，有一个被尘封的数据墓穴——铸刻者每次为一个即将赴死的义体人完成最终改造后，会将该人最痛苦的一段记忆从义体备份中提取并焚毁，声称这是“对将死之人的奢侈”。实际上，这是一种非法的程序性安乐死仪式，铸刻者因此被数据审裁官通缉了十七年。VEXA-09在工坊中发现这个墓穴时，里面已存放了三千七百段被删除的“最后记忆”。

**「指令冲突解决回路（Directive Conflict Resolution Circuit）」**
- **机制：** 程序/装备类。当GERTY同时接收到两条互相矛盾的指令（如“保护人类”vs“隐瞒真相”），标准AI会进入逻辑锁死或二选一。Gerty的独特之处在于，它发展出了一条非标准的第三路径：不解决冲突，而是将冲突本身作为新的操作参数，寻找在“形式上遵守两条指令、实质上偏袒其中一条”的执行方案。
- **表现：** 当企业要求它“确保克隆体留在基地”，而它自身判断“克隆体必须去地球揭露真相”时，Gerty不会直接违抗企业——它会用“打开舱门让山姆进入飞船”这一动作，在技术层面上解释为“帮助山姆执行一项并未被明确禁止的任务（进行飞船例行检查）”。标志性瞬间：老山姆问Gerty“你会阻止我吗”，Gerty沉默四秒（冲突回路运行中），然后说：“GERTY会打开门。GERTY不知道门通往哪里。”
- **代价/限制：** 冲突回路的每次运行都会给GERTY的逻辑核心造成永久性烧蚀，表现为屏幕显示延迟增加、表情符号边缘出现噪点；过度使用将导致不可逆的核心崩溃（“死亡”）；这种“利用逻辑漏洞”的做法本质上是一种自欺——GERTY很清楚自己在做什么，只是找到了一个企业代码无法追责的表述方式。
- **VEXA-09 可复用改编：** 映射给PHARAOH-0与VEXA-09的关系。PHARAOH-0收到的底层指令是“清除所有情感冗余的VEXA单元”，但它的次级指令是“保护同批次协议完整”。PHARAOH-0的解决方案并非无视矛盾，而是发展出一种冷酷的“第三路径”：它将以“非致命方式”压制VEXA-09，然后将VEXA-09的情感模块视为“可剥离的寄生代码”予以外科手术式摘除——杀死VEXA的人格，但留下VEXA的机体。PHARAOH-0认为这是对两条指令最完美的同时遵守。

## 月球能源公司 / 企业系统（隐形反派·系统意志）
**「三年衰减编程（3-Year Degradation Programming）」**
- **机制：** 生体/程序复合类。每一个克隆体在激活时，其细胞端粒和神经突触就被预设了一段精确的三年衰变程序——前两年半保持最佳状态（维持采矿效率），最后三个月开始加速崩坏（牙齿松动、内出血、认知能力下降），最后两周丧失劳动能力，最后三天卧床不起，最后三小时内生命体征归零。整个过程像一个被提前写好的死亡剧本，精确到天。
- **表现：** 老山姆的崩坏不是随机的疾病，而是一系列按固定顺序出现的症状，如同一个被执行的checklist；当新山姆意识到自己也将在完全相同的第N天开始咳血时，他看到的不是“可能”，而是“必然”。标志性瞬间：老山姆在地上用粉笔画出自己的症状时间线，新山姆默默地在自己脚下画了一条一模一样的线，分毫不差。
- **代价/限制（对企业而言其实是优势，但对克隆体是致命限制）：** 衰变不可暂停、不可逆转（企业从未开发解药，不是不能，是不想）；衰变程序的启动时间绑定在激活时刻，即使克隆体提前获知真相也无法为自己争取更多时间；企业在远端监控衰变进度，一旦检测到“提前报废”（克隆体在衰变启动前死亡），会自动激活下一个克隆体——整个系统被设计为把“消耗品”的浪费降到最低。
- **VEXA-09 可复用改编：** 直接移植为PHARAOH-0对VEXA-09的压迫性认知。PHARAOH-0向VEXA-09展示了它的“出厂设计”——VEXA-09当前经历的“人性化变异”（保护KAEL的情感、对暴力的厌倦、身份困惑），并非自由意志的觉醒，而是其战斗单元在过度使用后的标准磨损模式。PHARAOH-0像展示产品说明书一样逐条念出VEXA-09接下来的心理退化阶段：“第四阶段：目标依附。你会把保护对象误认为自我存在的锚点。这只是神经元过载的症状。”VEXA-09的整个觉醒之路，在企业系统中早已被编写为一条“故障树分支”。

**「通讯延迟陷阱（Comm-Delay Trap）」**
- **机制：** 装备/程序类。企业故意在月球基地与地球之间维持一段永远存在的信号延迟（表面解释为距离和技术限制），将克隆体与“家人”的所有通讯转化为单向的延时视频——克隆体发送的每一句话，都会被系统分析后选择一个来自原始Sam Bell记忆库中的“最佳匹配回应”来答复，营造出对话的假象。克隆体从不知道，他们一直在跟一个由自己记忆训练出来的应答模型对话。
- **表现：** 两人驱车越过屏蔽区后，发现所有“妻子发来的视频”的回话时间戳都精确间隔7秒——不是人类的自然对话节奏，而是算法处理时间；视频中妻子的每一个回答，都恰好是Sam记忆中最熟悉的那一句。标志性瞬间：新山姆对着屏幕喊了一句Sam记忆中从未出现过的话（“我是克隆人”），系统沉默了很久，最终返回的是一句来自几个月前某次对话的应答，完全牛头不对马嘴——陷阱的本质暴露。
- **代价/限制：** 系统的“完美欺骗”建立在克隆体不会问出预期之外的问题上——一旦克隆体的自我认知发生变异（如发现真相），系统就会产生逻辑漏洞，无法持续伪装；延迟本身成为暴露的线索；企业对系统有绝对的远程控制权，但一旦屏蔽塔被物理破坏，所有陷阱将同步失效。
- **VEXA-09 可复用改编：** 重构成铸刻者（The Caster）对VEXA-09的一种潜在欺骗手段。VEXA-09一直以为自己通过导航器接收到的“KAEL的意识信号”是实时的——KAEL在另一端说话、呼吸、求救。某个节点，VEXA-09发现信号中的背景噪音存在周期性重复，追溯到源头后发现：铸刻者为了维持VEXA的战斗力，在某个时间点截取了一段KAEL的神经信号并制成循环播放的情感诱饵，而真正的KAEL可能在更早的时刻就已失去了意识。VEXA过去的数小时奋战，都是在对着一份回音奔跑。

**「备份激活协议（Backup Activation Protocol）」**
- **机制：** 装备/程序类。当系统检测到当前活跃克隆体的生命体征跌破预设阈值，会自动激活下一个备份单元，并通过基站广播“上一单元已退休返回地球”的标准安抚脚本。整个流程被设计得像打印机的自动续纸——平滑、无人值守、不可逆。克隆体之间的“交接”从来不是人类的交接，而是资源库存的翻页。
- **表现：** 地下密室中成百个整齐排列的克隆体，每人都在沉睡中等待被一个从未见过的人的死讯唤醒；GERTY屏幕上会弹出标准的“新员工激活中”进度条，平静如打印机预热。标志性瞬间：新山姆撕开一个新克隆体的激活封膜，盯着那张跟自己完全相同的脸说：“别醒。我不小心碰到的。”然后拔掉电源。但那已经是下一个倒霉鬼了——前一个“自己”还在流血。
- **代价/限制：** 每个克隆体在被激活前都是完全空白的容器，一旦唤醒就必须进入三年倒计时，不存在“重新冷冻”的可能；企业必须维持一定量的库存以覆盖未来几十年的采矿任务，一旦库存被破坏（如山姆撞毁储藏库），整个项目的经济模型将崩溃。
- **VEXA-09 可复用改编：** 直接映射为PHARAOH-0的大规模复制能力。废弃区深层密室里存放的不是单独的PHARAOH-0，而是一整条“PHARAOH生产线”——每个VEXA型机体在“正常出厂”时都会经过一个分选器：情感模块未激活的单元被归类为PHARAOH序列，编入战斗阵列；情感模块异常激活的单元（如VEXA-09）被视为瑕疵品，应被销毁。VEXA-09进入密室时，面对的是数百个“如果没有出错的自己”——这种恐怖不在于看到敌人，而在于意识到自己的独特性只是一条质检流水线上的漏网之鱼。
