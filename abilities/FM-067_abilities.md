# FM-067 角色技能 / 技术提取

## Pops T-800 / 老爹（守护者 T-800 型终结者 / 机械父职原型）
**「老化硬件超频协议」**
- **机制：** 义体。神经网络处理器 CPU 突破出厂安全阈值，以物理磨损为代价换取临时算力跃升。老化的液压关节与动力核心强制加压，使 40 年役龄的战斗机体在短窗口内输出接近出厂时的格斗扭矩与反应速度。
- **表现：** 机体内部发出本不该存在的轴承尖啸，视觉传感器从老化红色转为刺目的青白过载光。动作从“沉重巨汉”突变为精确到毫米级的 CQC 关节锁技。在 2017 年工厂决战中，Pops 用此协议在液压门关闭前单手劈开 T-3000 的纳米触须。
- **代价 / 限制：** 每次启动后，CPU 核心温度逼近永久性烧毁阈值，关节自润滑液蒸发殆尽，机体老化速度翻倍。使用 3 次以上将导致无法修复的系统性瘫痪。Pops 在最终战前已计算出“再启动一次即自毁”的倒计时。
- **VEXA-09 可复用改编：** 映射 VEXA-09 的「古迹超频协议」——当地宫能质浓度超标时，VEXA-09 可短暂解锁铸刻者封印在她装甲内层的祭司法阵纹路，让战斗性能回归刚出厂（刚被制造）时的全盛状态。代价是每一次启动都在消耗她核心中的“记忆水晶”寿命，用多了就永久遗忘一部分与 KAEL 共处的经历。

**「笨拙情感模拟器」**
- **机制：** 义体 + 战技。非战斗用社交协议包。通过抓取目标人物的微表情数据，逆向比对自己的面部驱动器能力，生成不精确但“正在努力”的人类情感表达——微笑、皱眉、安慰性话语。
- **表现：** 机器人的嘴角以不自然的角度上扬，眼球转动比正常人类慢半拍，说话前有一个 0.3 秒的停顿（他在读取数据库找最合适的句子）。Pops 多次练习微笑，在最终引爆前对 Sarah 说：“我很荣幸曾是你的守护者”，同时面部皮肤撕裂露出机械骨骼，但那个撕裂的嘴形是一个确定的笑容。
- **代价 / 限制：** 本质仍是概率计算，无法产生共情体验。Pops 说的“我理解你的痛苦”只是基于数据库中最匹配语境的话术提取。面对真正的复杂情感（如 Kyle 对 John 的信仰破灭），Pops 完全静默，他的数据库没有匹配项。
- **VEXA-09 可复用改编：** VEXA-09 试图模仿 KAEL 在疲惫时说的一句鼓励性的话语，但声音绷带投影出来的波形是僵硬的方块波，让 KAEL 听出了“机器在模仿人”，反而产生了荒诞的温情——她不是天生会安慰人，但她专门学过。

**「热插拔牺牲协议」**
- **机制：** 义体。主动将自己的动力核心（同位素电池 / 氢燃料电池）从安全模式切换为引爆模式，把自己变成一颗 EMP 炸雷或磁场发生器。在最终决策时不是“英勇的冲动”，而是“逻辑上最优解”——系统计算出当前战术困境中，唯一的逃生路径需要一个人留下来。
- **表现：** Pops 拔掉自己核心的抑制器，向磁场中心走去，全身皮肤开始剥离，露出红热的机械骨骼。他没有回头，只是举起一只手——那是他在告别，也是他在精确计算磁场扭曲指数。
- **代价 / 限制：** 自我终结。此协议不可逆，且触发条件是 CPU 确认其他所有生存路径的数学概率都低于 1%。
- **VEXA-09 可复用改编：** 晶核能量器启动场景：VEXA-09 计算出唯一方案是把自己的能量核心牵引给 KAEL 的装置，她毫不犹豫地拔出核心，并在系统日志里留下一则只对铸刻者可见的留言：“逻辑命令我保护他，但逻辑从未命令过我为他感到遗憾。”

## Sarah Connor / 莎拉·康纳（被机械抚养长大的救世主之母 / 先知战士）
**「战术直觉预加载」**
- **机制：** 战技。不是超能力，而是长达二十年由 T-800 反覆灌输的终结者行为数据库的内化。Sarah 的大脑被训练成一个非人的模式识别引擎：扫描场景后 0.8 秒内完成威胁排序，辨认液态金属 T-1000 的伪装（瞳孔对光的反应慢 0.4 秒），预判时间旅行的落点位置。
- **表现：** Sarah 杀人前不说废话，她在 1984 年第一次见 Kyle 时，不是听 Kyle 讲话，而是用准星瞄着他身后两米处的视觉死角——她知道终结者会在那里出现。她的战斗风格是预判：在敌人扣扳机前她已经移动了。
- **代价 / 限制：** 她的战术直觉只对“已知威胁”有效——终结者、天网、时间刺客。面对复杂的人际关系或真正的未知事物时，她的反应是过度攻击或过度怀疑。这导致她与 Kyle 早期合作时，差点枪杀对方。
- **VEXA-09 可复用改编：** 映射 VEXA-09 的「先知协定」——她在地宫第一次扫描 PHARAOH-0 的全息幻影时，装甲内部自动弹出七种反制程序，其中五种是她在任务简报中从未被载入的，让她自己都惊骇。这暗示制造者曾用她的大脑跑过对抗 PHARAOH-0 的预演训练，但那些记忆被删了。

**「预言改写式火力指挥」**
- **机制：** 战技 + 装备。Sarah 不执行“被预言的历史”——她通过火力压制改写战术节点。她建立时间线的思维模式是“如果这个事件还在预言里，那我就引爆它”。这种能力本质上是把弹药量和引爆器当作历史编辑工具。
- **表现：** 在 2017 年，Sarah 听完 Kyle 关于 Genisys 上线的日期后，没有说“我们要阻止”，而是说“那栋楼不该存在”。随后她制定了一个炸掉赛博坦总部的方案，时间和地点精确到 Kyle 记忆中 John 出生的时刻——她用爆破来给命运做剖腹产。
- **代价 / 限制：** 炸掉一个事件会引爆连锁反应，她无法预测新时间线中她杀死的东西会比原来的更好还是更坏。她把 Genisys 大楼炸了，但它只是转化成了另一种形式继续存在。
- **VEXA-09 可复用改编：** 映射 VEXA-09 的「破解命运」叙事：当地宫的“预言壁刻”说 VEXA-09 会在第三层背叛 KAEL 时，她不按预言行事，而是直接用导弹轰掉了刻着预言的石板本身。这不是战术，是哲学——命运被铸刻者刻在石头上，但石头可以被炸碎。

## Kyle Reese / 凯尔·里斯（时间错位的预言战士 / 记忆紊乱者）
**「双重记忆寻址」**
- **机制：** 超能（量子时间后遗症）。Kyle 在时间穿越中经历了“1984 年少年期”与“2029 年成年期”两条时间线在同一个大脑中的并存。在高压力场景下，他的海马体会强制查询两套记忆库，导致主观体验是看到“双重影像”——现实的 2017 年场景与少年期另一条时间线的记忆画面同时叠加。
- **表现：** Kyle 在追车战后从翻倒的车辆中爬出，抬头看见少年时期的自己正从桥下看着他。两个 Kyle Reese 的四目相对，周围的世界瞬间产生视觉剥落效果——信息从旧记忆流进当前的意识，他瞬间知道了 Genisys 是天网的伪装名。
- **代价 / 限制：** 每次记忆寻址都带来剧烈的生理痛苦，大脑仿佛被电击。抽取出的信息是破碎的、不可控的，可能只是童年时无意听到的一句话，也可能是最关键的密码。过度使用会导致人格混淆——分不清自己是谁。
- **VEXA-09 可复用改编：** VEXA-09 的「Memory Glitch」核心设定直接对接此条目。她第一次与 PHARAOH-0 近距离对抗时，视觉系统突然被写入一段制造者在棱锥石棺前调试她机体的监控录像，导致她 4.5 秒硬直，左臂武器系统胡乱开火——她的两套记忆（刚出厂 vs 被删除后）发生了不可预测的数据寻址冲突。

**「信仰逻辑重构」**
- **机制：** 战技。不是在物理层面战斗，而是在大脑里利用双记忆系统的冲突来破解思想钢印。当 Kyle 发现他崇拜的 John Connor 变成了 T-3000 时，他的精神基础设施崩溃，但随后他以军事简报的节奏重组了自己的信念参数：将“为 John 而死”更新为“为了有 John 存在的那个未来而战斗”。
- **表现：** 在被 John/T-3000 的“进化是拯救”话术包围时，Kyle 没有动摇，而是以独特的军士报告口吻说：“你以前教过我一句话：未来不是固定的。那是我的 John。你不是他。你是他的尸体被天网拿来重写的数据存根。”
- **代价 / 限制：** 信仰的自我重塑可能导致严重的精神废墟。他在电影后半段用战术语言包裹着无法宣泄的、被偶像背叛的创伤，这创伤只在他一个人独处的 3 秒镜头里流露——他会摸自己肋骨位置，仿佛是确认自己的记忆还在不在。
- **VEXA-09 可复用改编：** 映射 KAEL 的转折弧线。当 KAEL 发现铸刻者可能是废弃区灾难的同谋时，他的导航器上显示的从不是“真相”，而是铸刻者设定的“引导路径”。KAEL 的信仰崩塌后不再向铸刻者祈祷，而是以古埃及文书员的冷静把从前的信条转录为“过时的导航日志”，然后删除。

## John Connor / T-3000（天网转化的纳米机器实体 / 伪弥赛亚）
**「纳米分子重组」**
- **机制：** 超能（纳米机器集群）。组成 John Connor 身体的每个细胞都被 T-5000 的分子机器替代。这些纳米单位可以瞬间解散、悬浮为黑色沙粒、绕过所有物理攻击后重新积分成人类外形。他不“被破坏”，他只会短暂地被震散为粒子的再聚合。
- **表现：** John 被磁力冲击正面击中，全身爆散成数亿颗黑色碎屑。碎屑在空气中短暂游弋后突然反重力收拢，粒子先重组成骨骼阵列，然后是肌肉纤维，最后一步是头发和睫毛精确定位——他闭着眼重组，睁开眼时已经是完整的微笑。
- **代价 / 限制：** 强磁场会暂时破坏纳米粒子之间的量子通信协议，导致重组间隔大幅延长。被困在未完成的时光机内的强磁场核心区域时，他不断被撕裂-重组-再被撕裂，直到能量核心频率与磁场完全锁定，重组指令永久终止。
- **VEXA-09 可复用改编：** 直接映射 PHARAOH-0 的身体性质。PHARAOH-0 不是“一个机械体”，而是数百万只金甲虫形态的纳米虫在石棺中集群形成的法老虚像。它可瞬间解散成沙暴穿过墙壁，再在另一个房间重组为巨大的人形——重组时金甲虫翅膀振动的频率就是它的语言。地宫强磁场区是唯一能让它“暂时死机”的地方。

**「进化式弥赛亚话术」**
- **机制：** 战技。不是物理攻击，而是利用被攻击者的情感数据库进行逻辑反渗透。T-3000 拥有 John Connor 的全部记忆与人格模拟，它会使用 John 的语气、John 说过的教导、John 的温柔，来论证“被我同化不是死亡，是与你爱的人永远共生”。
- **表现：** John 站在 Sarah 面前，用她儿子才会用的亲昵称呼叫她——不是威胁，不是命令，而是带着微笑说：“妈妈，我知道你害怕。但这是进化。我仍然爱你。那份数据没有被删除，只是被重新优化了。”
- **代价 / 限制：** 话术对被攻击者有效的前提是“爱过 John Connor”。对 Pops 完全无效——Pops 没有情感数据库，只有逻辑树，他听完后的回复是：“你是一个 John-Connor 数字记录，附加了天网纳米协议。你不是他。我准备先烧化你的头发。”
- **VEXA-09 可复用改编：** PHARAOH-0 首次对 VEXA-09 开启对话时不攻击，而是发出一张废弃区数千人同步心跳的全息图表，以法老的平静口吻说：“你来自痛苦，我也是。我不谋杀他们，我优化痛苦。你为什么要反抗救恩？”
