**「黑暗中焊光揭幕」**
- **原片片段：** 洞穴中 Mark I 装甲首次启动，蓝色反应堆点亮托尼胸口，镜头从暗无光到第一缕蓝光渗出
- **镜头序列：** 全黑画面(2s) → 极微弱的蓝色光点从画面中心渗出(广角固定，焦外全黑) → 光晕逐渐扩大照亮托尼胸口轮廓(缓慢拉焦，从模糊到锐利，0.8s) → 手持镜头轻微呼吸式晃动，跟随蓝光上摇至托尼脸部(中近景) → 切焊枪火花溅入画左(0.3s快切) → 切尹森侧脸被蓝光照亮(中景，固定) → 回切托尼正面，反应堆完全亮起，整张脸从下往上被蓝光勾边(升格，0.5x) → 全景拉出，洞穴墙壁上投射巨大装甲阴影
- **为什么有冲击力：** 从绝对黑暗到蓝光渗出的视觉钩子制造“诞生感”；升格落幅让技术时刻获得宗教仪式般的重量；阴影投射暗示小型光源即将释放巨大力量，用景别落差完成从私密到宏大的情绪跳跃
- **VEXA-09 可复用模板：** `晶核能量器激活=全黑2s→晶核第一缕光渗出→拉焦锐化→手持上摇至VEXA-09护目镜碎裂处→升格落幅蓝光漫溢全屏→俯拍拉出地宫祭坛全景，墙壁投射VEXA-09巨大装甲阴影` 适配晶核能量器激活、反应用堆首次点亮、任何“从无到有”的能量觉醒场景

**「坠落主观翻滚长镜」**

- **镜头序列：** 广角低机位仰拍楼顶边缘，铁霸王挥臂，托尼从画左上被击飞出画(24fps) → 镜头瞬间切换为固定在托尼胸甲上的GoPro式主观视角(超广角鱼眼，画面出现畸变弯曲的地平线) → 镜头随装甲在空中翻滚：天空→城市灯光→天空→地面→天空→地面(每次翻滚约0.8s，无任何剪辑，完全依赖实拍或CG连续运算) → 翻滚中碎片/玻璃/烟尘不断从画面前景掠过(物理真实感强化) → 托尼的UI在画面四角间歇闪烁红色警告，每次翻滚到“地面朝上”时叠加一层红色滤镜 → 第5次翻滚后，画面突然稳定(装甲恢复控制，助推器点火) → 镜头从主观POV切为外部无人机跟拍视角(从下往上仰拍，装甲拖着烟雾重新拉升)
- **为什么有冲击力：** 无剪辑的长镜头让观众无法从眩晕中逃脱，强制体验坠落失重；超广角鱼眼畸变+旋转地平线制造生理级不适；翻滚与红色警告同步叠加增强“濒死”节拍感；从失控翻滚到突然稳定的镜头切换提供巨大释放
- **VEXA-09 可复用模板：** `废弃区坠落=仰拍被击飞出画→胸甲GoPro主观超广角鱼眼→连续翻滚无剪(天空→废墟光→天空→地宫深处)→碎片掠过前景→每翻至“地面朝上”叠加红光→第N次翻滚后稳定(晶核反重力激活)→切外部仰拍拉升` 适配VEXA-09被PHARAOH-0击落、电梯井坠入地宫、任何高空坠落失控时刻
