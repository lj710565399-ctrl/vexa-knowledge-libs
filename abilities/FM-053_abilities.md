# FM-053 角色技能 / 技术提取

## Jack Sparrow（疯癫海盗船长）
**「神弃罗盘·欲念导航」**
- **机制：** 装备类（附魔遗物）。罗盘不以地磁定位，而是指向持有者“当下最渴望之物”的方向。该物可以是人、物品、地点或抽象概念。原理疑似绑定持有者的潜意识欲望场，不受物理遮蔽干扰。
- **表现：** 罗盘指针在无人握持时静止，一旦触碰皮肤即开始旋转，最终锁定一个方向。视觉上指针会有微弱的金色拖尾。标志性瞬间：杰克在摇摆不定时罗盘疯狂旋转，暴露其内心矛盾；或罗盘指向一个他自己都不愿承认的渴望。
- **代价 / 限制：** ⚠️持有者必须有明确的欲望，否则罗盘失效。更致命的是——若持有者“背叛”罗盘（放弃自己真正的渴望），罗盘会释放其封印的“最深的恐惧”。萨拉查正是因此被释放。本质上是一个欲望契约，不能违约。
- **VEXA-09 可复用改编：** 可映射为 **晶核导航器**——一个赛博义体插件或古埃及圣物，能绕过物理导航直接指向目标人物的“Ka（灵魂意志）”。给 VEXA-09 使用：她用它追踪 PHARAOH-0 时，导航器突然指向自己，揭示她潜意识里想找回被删除的记忆人格。代价：每次使用都会在赛博空间留下欲望印记，被铸刻者反向追踪。

**「疯癫闪避·醉步流」**
- **机制：** 战技类（直觉型体术）。看似毫无章法的踉跄步伐，实则是一种高度不规则的随机运动模式，使对手无法预判行动轨迹。依赖极端的环境感知和即兴反应，而非训练有素的肌肉记忆。
- **表现：** 身体重心不断偏移，手脚动作与注意方向完全分离。敌人瞄准时，他刚好弯腰捡帽子；炮弹袭来时，他恰巧被绊倒。视觉上呈现为“好运的连续巧合”。标志性瞬间：在多人围捕中，所有攻击恰好擦身而过，他本人一脸“发生了什么”的茫然。
- **代价 / 限制：** ⚠️不可主动复现。一旦他试图“认真战斗”，该技能完全失效。本质上依赖“放弃控制”的心理状态，酒精或极端恐惧是其触发器。清醒且自信的杰克反而是最弱的杰克。
- **VEXA-09 可复用改编：** 可映射为 **故障步法（Glitch Step）**——一种赛博神经系统的非稳定态，义体运动控制被人为注入随机种子算法，使动作完全脱离战斗预测模型的捕捉。给 KAEL 的反抗军线人使用：一个被实验过的废品赛博格，他的义体有永久性运动神经 glitch，反而因此无法被 PHARAOH-0 的预判战斗 AI 锁定。

**「诡辩契约·话术脱缚」**
- **机制：** 社会工程类（语言 / 认知操控）。通过在谈判中引入多层嵌套的矛盾承诺、模糊措辞和语义陷阱，使对方在逻辑上被自己的话困住，从而在看似已达成协议的情况下实现完全不同的结果。
- **表现：** 大量使用“取决于你对‘是’的定义”“从某个角度看”等模棱两可的措辞，配合高速语流让对方来不及逐条解析。签订契约时总会留下一个对方忽略的漏洞。标志性瞬间：欠下血债却用“我的船就是你的船——但船在被偷”完成逻辑闭环。
- **代价 / 限制：** ⚠️信用破产。长期使用导致所有盟友和敌人都不会信任他的任何承诺，重要合作需要物理要挟才能推进。他自己也陷入无法分辨真假承诺的认知混乱。最严重时，连戴维·琼斯级别的超自然契约都可以被他“话术规避”，但代价是终生被追杀。
- **VEXA-09 可复用改编：** 可映射为 **协议蠕虫（Protocol Worm）**——一种基于古埃及祭司法庭辩论逻辑编写的赛博契约病毒，能发现任何电子协议中的逻辑漏洞并生成豁免条款。给铸刻者阵营的某个叛逃文书官使用：他帮 VEXA-09 逃脱了与 PHARAOH-0 签订的强制执行协议，但蠕虫反噬，他的语言中枢被永久污染，说任何真话都会伴随一句谎话。

## Captain Salazar（亡灵船长 / 西班牙猎盗者）
**「亡灵船·追猎场域」**
- **机制：** 超能 / 诅咒类（集体怨念实体）。萨拉查及其船员死于魔鬼三角区的爆炸，怨念与海洋本身融合，形成“沉默玛丽号”这一半船半尸的诅咒聚合体。船体可无视物理法则行动——包括踏浪行走、舰首劈开水面如犁地、船体自我修复。船员以残缺亡灵的形态存在，与船共享意识。
- **表现：** 沉默玛丽号的船首如张开的下颌，航行时整艘船呈半潜姿态，突然从水下破浪而出。船员缺失的身体部位被灰烬和海水替代，行动时留下灰痕。标志性瞬间：沉默玛丽号从水下直接“咬住”敌人船舰，亡灵船员如潮水般涌上敌船，灰烬弥漫如海底坟墓。
- **代价 / 限制：** ⚠️无法上岸。诅咒场域局限于海水触及之处，一旦目标逃入陆地，追猎即中断。萨拉查本人无法离开船体触及范围之外的海水。更致命的是——诅咒的唯一解除方式是三叉戟被毁，而三叉戟一旦被毁，亡灵形态连同执念将一并消散，不存在“解咒后继续活”的选项。
- **VEXA-09 可复用改编：** 可映射为 **怨念聚合体·奈赫布号**——一艘由古埃及墓穴中殉葬者意识融合而成的赛博幽冥舰，船体材料是凝固的纳米灰与记忆合金，能在数据海中潜行并突然实体化。给 PHARAOH-0 的海上执法部队使用：PHARAOH-0 将反叛者的人格副本灌入船体，用他们的怨念驱动战舰，但每次追击时必须播放叛徒们生前记忆作为导航燃料。

**「复仇嗅觉·血仇追踪」**
- **机制：** 超能类（诅咒感知）。萨拉查对“海盗”这一概念拥有超自然的嗅觉，尤其是与他有直接血仇的杰克·斯派罗。不依赖任何线索或情报，怨念本身会自动牵引沉默玛丽号向目标方向航行。距离越近，萨拉查的亡灵形态越不稳定（灰烬加速崩落），提示复仇将至。
- **表现：** 萨拉查站在舰首闭眼，空气中飘散的灰烬粒子开始定向流动，船体随之转向。他从不说“找到他”，而是说“我闻到了”。标志性瞬间：在茫茫大海上，沉默玛丽号无视洋流和风向，直直朝一个方向突进，最终精准截住目标。
- **代价 / 限制：** ⚠️单线程执念。复仇嗅觉只对“仇恨目标”敏感，其他船只哪怕从旁边经过也视若无睹。萨拉查无法同时追踪多个目标，无法被战略性地分兵。且仇恨越强烈，越容易被仇恨对象设陷阱利用——杰克正是利用这一点将其引入魔鬼三角区。
- **VEXA-09 可复用改编：** 可映射为 **血债协议·人格锁定追踪**——一种古埃及的血魔法赛博化版本：将受害者的血样注入追踪者的赛博义眼，义眼会始终显示目标的方向和距离，但作为代价，追踪者的视野中其余一切都会失色为灰白。给 PHARAOH-0 的猎杀执裁官使用：他的双眼只能看到目标的红色热痕，但已无法分辨无辜者。

## Carina Smyth（卡琳娜·史密斯 / 天文学家）
**「星图解算·天文导航」**
- **机制：** 知识技能类（天文学 / 数学）。通过观测星体位置、结合精确时间（依靠自制精密计时器），在没有罗盘和海图的情况下计算出船只的经纬度和正确航线。本质是将星空视为一张加密地图，用数学进行解密。
- **表现：** 快速进行六分仪测角、查阅伽利略日记中的星表、心算球面三角。整个操作过程冷静高效，通常伴随对其他船员“迷信”的嘲讽式科普。标志性瞬间：在幽灵船追击、全员恐慌时，她独自在暴风雨中测算出一条“不可能存在”的安全航道，仅凭星光的几秒缝隙完成定位。
- **代价 / 限制：** ⚠️必须依赖可见星空。云雾、白天、水下环境完全失效。同时极度依赖精密计时器的准确性，计时器一旦损坏或误差累积，所有计算结果作废。此外，其天文知识在这一时代被等同于“巫术”，施用时需冒被处决的社会代价。
- **VEXA-09 可复用改编：** 可映射为 **盖布之眼·轨道解算器**——盖布（古埃及大地之神）信仰的赛博重释：利用人造卫星残骸和太空碎片的光反射模式，在赛博网络瘫痪时进行离线导航。给叛逃至反抗军的前铸刻者天象祭司使用：她能“阅读”太空垃圾的闪烁规律，但每次计算需关停所有赛博植入体以避免电磁干扰，此时的她彻底暴露且毫无防御。

**「计时器工程·精密制械」**
- **机制：** 装备制作类（精密机械工程）。独立设计并制作了能精确测量经度的航海计时器——这在 18 世纪是超越时代的尖端技术。该计时器不依赖摆锤（船上摇晃会失效），而是用发条与游丝结构维持恒定时基。
- **表现：** 一个手工打磨的金属怀表状装置，内部齿轮密布，在暴风雨和炮火中依然精准走时。她对这个装置的珍视程度超过生命——不仅是科学工具，更是她身份的唯一信物（生父留下的遗物线索）。标志性瞬间：计时器被敌人夺走时，她放弃逃命回头抢夺，用一句“That's a chronometer. It measures longitude.”在命悬一线时仍做科普。
- **代价 / 限制：** ⚠️手工精度有物理上限。长航程后累积误差会逐渐放大，需定期靠天文观测进行修正。且装置为孤品，一旦损坏无法在海上修复。更关键的是，它作为她“被遗弃的证据”，承载着巨大的情感脆弱性——敌人一旦识破其情感价值，可用它精准操控卡琳娜的行为。
- **VEXA-09 可复用改编：** 可映射为 **荷鲁斯之芯·以太时基振荡器**——一件融合古埃及计时圣物与赛博晶体振荡器的手工制器，能在赛博网络时间协议被污染 / 篡改时提供绝对独立的时间基准。这正是 VEXA-09 在废弃区深处发现的那件古物：她能修好它，因为它与自己的神经接口频率完美匹配——暗示她可能曾是这件圣物的铸造者，或被这件圣物选中。

## Hector Barbossa（赫克托·巴博萨 / 奸猾海盗船长）
**「海盗契约·话事权评估」**
- **机制：** 社会工程类（博弈论 / 本能谈判）。巴博萨能在任何三方或多方对峙时，以极快速度完成所有参与方的筹码评估，然后用一套包含“威胁列举”、“共同利益虚构”和“暂定联盟报价”的话术，将自己放置在不可替代的中间人位置上。
- **表现：** 声音洪亮而缓慢，每次开价前会用“Arr”给自己一秒的思考缓冲。双手长期按在枪和剑柄上，用微小的武器姿态调整来暗示每一次报价的诚意。标志性瞬间：在黑石岛三方对峙时，杰克有情报、萨拉查有武力、卡琳娜有解谜能力，而巴博萨在 60 秒内通过依次向三方分别承诺不同内容，成为唯一一个三方都“暂时不能杀”的人。
- **代价 / 限制：** ⚠️长期积累的背叛信用。巴博萨的结盟承诺有效期极短，所有合作方都默认他会在最后关头反水——这意味着他必须在每个合作方反水之前先下手。他的生存完全依赖“比别人早一秒背叛”，这一秒一旦失手就是死亡。此外，维持复数的虚假承诺会高度占用他的注意力，无法专心战斗。
- **VEXA-09 可复用改编：** 可映射为 **九柱神协议·多方博弈算法**——基于古埃及九柱神神话（九神各自为政又互相制衡）开发的赛博谈判 AI 插件，能在数据战中实时分析所有势力的利益矩阵并生成最优背叛时机。给反抗军的地下情报掮客使用：他的大脑被植入这套协议，每次谈判都能看到所有人的“预期背叛时间线”以倒计时形式浮现，但他自己也被算法奴役——如果不按最优时机背叛，协议会触发剧痛惩罚。

**「恶棍柔情·遗计自爆」**
- **机制：** 战技类（终极牺牲型战术）。在意识到自己已无法脱身时，巴博萨会切换到一种放弃了所有贪婪和算计的纯粹战斗姿态——以自身为饵，将自己和敌人一同拖入必死的陷阱。这一行为并非源于道德觉醒，而是源于一种被压抑的、只对血亲才会触发的保护本能。
- **表现：** 表情从一贯的奸猾突然转为平静，最后一句台词通常是一句收敛了所有表演的实话（“你是我的珍宝”）。动作变为不计伤害的抓取式突进，用剑锚住敌人身躯，借重力或环境致命物共同坠入深渊。标志性瞬间：一手持剑刺穿萨拉查，另一手抓住卡琳娜防止她坠崖，在确认女儿安全后松手下坠，全程没有一句对自己牺牲的渲染。
- **代价 / 限制：** ⚠️唯一的代价就是生命本身。此技能只能触发一次，且触发条件极为苛刻——必需是血亲处于无法被任何“交易”拯救的绝境。在漫长的海盗生涯中，巴博萨有无数次危险，但只有这一次，他的本能压倒了计算。本质上，这不是一种技能，而是一种“角色代码最深处的底层指令”，无法被训练、复制或提前计划。
- **VEXA-09 可复用改编：** 可映射为 **父权覆写·安普特协议**——安普特是古埃及神话中为亡者引路的女神，此处重释为一种赛博人格底层代码：当使用者的直系血亲生命体征归零时，其所有生存算法被强制覆写为一条自我献祭指令。给铸刻者阵营的某个高阶祭司使用：他毕生执行 PHARAOH-0 的清洗命令，直到自己的义体女儿出现在处决名单上——安普特协议启动，他抱着女儿跃入反应堆冷却池，用自己的赛博核心为女儿置换出最后一条生路。
