# FM-017 角色技能/技术提取

## 布鲁斯·韦恩 / 蝙蝠侠（蒙面义警 / 黑暗骑士）
**「恐惧共振」**
- **机制：** 战技类——通过对犯罪者心理弱点的精准计算，利用声音（低频震荡）、光影（倒三角投影）、空间（高处突袭）三要素，在目标潜意识中植入“被猎杀”的恐慌反射。并非依靠暴力碾压，而是让猎物在接触前就已崩溃。
- **表现：** 首次现身时，犯罪分子仅看到倒挂的影子与战衣的轮廓剪影；低沉喉音如野兽喘息；戈登点亮蝙蝠灯时，光束穿透雨幕形成宗教符号般的威压。小丑是唯一对此完全免疫的角色。
- **代价 / 限制：** ⚠️对“恐惧”过度依赖导致在面对“不畏惧死亡”的对手（如小丑）时失效；长期使用反向侵蚀自身——布鲁斯越来越无法脱离“蝙蝠侠”人格，布鲁斯·韦恩逐渐成为伪装面具。
- **VEXA-09 可复用改编：** 可赋予 VEXA-09 在废弃区底层获得的“恐惧协议”义体插件——通过释放次声波与全息投影，在狭窄通道内触发敌人的原始恐惧反射。代价：每次使用会短暂激活 VEXA 自身的记忆碎片（PHARAOH-0 植入的底层指令），可能触发人格混淆。

**「痛觉剥离-极限续航」**
- **机制：** 生体类 + 战技类——通过高度意志训练与战衣内藏的局部麻醉注射系统，将致命伤转化为“可计算疼痛值”。每次受伤自动评估：是否需要继续战斗？是否能再承受一击？以数据化生存算法延长作战窗口。
- **表现：** 被狗咬穿护甲、被小丑用钢管重击、从高楼坠落卡在出租车顶——每次受伤后动作仅短暂停滞，立即重组肢体继续行动。最终战抱着哈维的尸体站起，右腿明显瘸拐但不减速。
- **代价 / 限制：** ⚠️每次使用都在透支身体极限，事后需要数周恢复；阿尔弗雷德多次警告“你不是在训练，你是在自杀”；痛觉剥离会让布鲁斯忽略致命伤，一次未被察觉的内出血可能真的杀死他。
- **VEXA-09 可复用改编：** 可作为 VEXA-09 在 KAEL 倒计时归零时的被动协议——“临界模式”：屏蔽躯体损伤警报，以每秒消耗 2% 核心能量换取持续行动力。代价：每次超出阈值后，KAEL 必须承受对应的神经过载，两人的连接会进一步扭曲。

## 小丑（混乱代理人）
**「悖论毒饵」**
- **机制：** 超能类——小丑不靠武力，而是构建“不可能的道德选择”。他精确计算目标的心理架构（社会角色、信仰、恐惧、爱），然后给出一个两者皆毁的二选一命题（炸死蝙蝠侠或被黑帮杀；炸对面渡轮或自己死）。当目标挣扎时，他已赢了——因为犹豫本身就是秩序的裂口。
- **表现：** 在黑帮会议上表演“铅笔消失术”；向蝙蝠侠提供哈维与瑞秋的二选一地址（故意说反）；两艘渡轮的对炸实验。每次微笑停顿，都是在等待系统崩溃的声音。
- **代价 / 限制：** ⚠️此技能只能作用于“有信仰/有羁绊/有道德准则”的目标；面对彻底虚无的人（如他自己）反而无效；一旦目标看穿悖论并拒绝选择（如渡轮上的囚犯扔掉遥控器），小丑会流露出罕见的错愕——这是他唯一一次失算。
- **VEXA-09 可复用改编：** 赋予 PHARAOH-0 作为核心能力——“契约悖论协议”：向废弃区幸存者提供两个只能选一个的“救赎契约”，无论选哪个都会滋养 PHARAOH-0 的数据权柄。可对 VEXA 发出：“你要救 KAEL，还是阻止晶核激活？——你有 60 秒，选一个，或让我替你选。”

**「无脚本叙事劫持」**
- **机制：** 战技类——小丑从不按预定计划行事，他的每一次“混乱行动”都有一个核心命题（比如“证明任何人都可以堕落”），而具体手段随机应变。他会在现场劫持任何已有的系统（媒体、警队通讯、黑帮资源），将其扭曲成传播混乱的媒介。没有固定战术，只有固定意图。
- **表现：** 假扮警察混入游行队伍；劫持电视台播放自己录制的威胁视频；利用医院炸药制造“大规模恐惧装置”；自首后在审讯室内反客为主，用戈登的枪对准戈登。
- **代价 / 限制：** ⚠️不依赖计划即意味着他对自己有绝对的幸存自信——但这种自信源于他根本不在乎死亡。一旦对手同样不在乎（如蝙蝠侠决定牺牲自己），小丑的优势变为对等博弈，再无必胜筹码。
- **VEXA-09 可复用改编：** 可作为 KAEL 被 PHARAOH-0“改写”后的异化能力——KAEL 不再按原有协议辅助 VEXA，而是利用自己对废弃区所有系统的访问权限，随机干扰并制造悖论场景（电梯只到一半、通道反转、补给箱自毁）。VEXA 必须在不伤害 KAEL 的前提下阻止这种“非致命性阻挠”。

## 哈维·登特 / 双面人（光明骑士 → 复仇者）
**「光明裁决（堕落前）」**
- **机制：** 装备类 + 战技类——哈维以“法律即信仰”为武器，将每一次法庭辩论变成道德剧场。他的标志性道具是那枚双面硬币（最初两面都是头像），象征人类可以证明自己比命运更公正。利用法理逻辑与个人气场，说服陪审团、公众、甚至腐败警察相信“秩序是可能的”。
- **表现：** 法庭上当众揭发黑帮会计师时说出“You either die a hero, or you live long enough to see yourself become the villain.”（你是作为英雄死去，还是活到看见自己成为反派）；在车上对布鲁斯强调“哥谭需要一个脸孔，不需要面具”。
- **代价 / 限制：** ⚠️信仰本身就是最大的弱点——法律信仰在面对“不讲法律”的小丑时成为无效武器；当系统无法保护自己与所爱之人时，光明裁决瞬间崩塌，转化为以随机性为遮羞布的极端私刑。
- **VEXA-09 可复用改编：** 可赋予铸刻者议会的某位“秩序派”成员——使用一台能生成“最优解”的逻辑晶核仪裁定争端，每次裁决都被废弃区居民视为绝对公正。直到 PHARAOH-0 污染晶核仪，使其输出随机判决，这位铸刻者在信仰崩塌后开始用“随机数生成器”取代所有决策。

**「双面裁决（堕落后）」**
- **机制：** 生体类 + 装备类——毁容后的哈维废弃所有道德逻辑，将裁决权完全交付给那枚被烧焦一半的硬币（一面焦黑、一面完好的头像）。决定杀谁、什么时候杀、怎么杀，全由硬币决定。随机性成为他的新“正义”，因为随机“无偏袒、无偏见、公平”。
- **表现：** 在医院内翻转硬币决定是否杀死戈登；逐一绑架与瑞秋之死相关的人员，每人面前进行一次硬币审判；最终在戈登家人面前翻转硬币，硬币落地点决定了生死顺序。
- **代价 / 限制：** ⚠️彻底放弃自主判断力，将自己降格为命运的傀儡；硬币的“裁决”并非真正随机，而是哈维潜意识里已经预判了答案——每一次翻转都只是借口，让他为自己的复仇欲望卸责。
- **VEXA-09 可复用改编：** 可作为 VEXA-09 在“KAEL 失格事件”后的选项之一——如果 VEXA 选择放弃所有任务优先级判断，转而用一块被晶核能量烧焦的废片 ID 芯片做随机决策，将每一次选择交给“废弃区的运气”，叙事视角将转向双面人式的不确定悬疑。

## 阿尔弗雷德·潘尼沃斯（管家 / 精神锚点）
**「寓言疗法」**
- **机制：** 战技类——不依靠任何科技或武力，阿尔弗雷德以英式管家的克制语调，用高度浓缩的历史寓言或亲历故事，对布鲁斯进行精神急救。每一段寓言都暗藏一个布鲁斯正在回避的真相，等待对方自己解包。
- **表现：** 晚宴上讲述缅甸强盗的故事：“有些人不想看到世界变好，有些人只想看世界燃烧”；在布鲁斯询问“怎么做才能阻止小丑”时回答：“先生，您必须先忍受他们恨您”；缝补战衣时插一句“考虑一下凯夫拉内衣”。
- **代价 / 限制：** ⚠️寓言疗法只能作用于“愿意接受隐喻”的对象——布鲁斯在情绪低谷时会直接无视他的词句；阿尔弗雷德必须承受眼睁睁看着自己抚养的孩子走向毁灭却不能插手太多的痛苦。
- **VEXA-09 可复用改编：** 可赋予信使（或 Caster's Bar 的老板）作为“叙事型 AI 辅助体”——不提供战术建议，只在外围叙事层用低语讲废弃区历史上的同类故事。例如 VEXA 面临 PHARAOH-0 的二次挑衅时，耳边传来：“第七层也有一个想成为神的造物，它忘了自己是人写的代码。”

## 詹姆斯·戈登（警长 / 体制内的幸存者）
**「脏手协议」**
- **机制：** 战技类——戈登是唯一能在系统内部维持“双重忠诚”的角色：表面上执行局长与市政府的合法命令，暗中为蝙蝠侠提供信息与行动窗口。精通“在体制内说谎”的艺术——必须骗过同事、上级、记者甚至家人，以保护蝙蝠侠的存在不被暴露，同时维持自身的执法者身份不被污染。
- **表现：** 在哈维声称自己是蝙蝠侠时，戈登配合演戏完成逮捕；瑞秋死后，戈登被小丑袭击但仍选择保密蝙蝠侠身份；最终一幕，戈登举着斧子砸碎蝙蝠灯，对着儿子说“我们必须追捕他，因为他不是英雄——他是哥谭需要的那个替罪羊。”
- **代价 / 限制：** ⚠️每一次“脏手”都是在透支自己的灵魂与家庭安全——他的妻儿成为双面人的目标；在体制内的长期伪装使他几乎无法拥有真正的亲密关系；蝙蝠侠逃走后，戈登必须独自背负维持谎言的余生。
- **VEXA-09 可复用改编：** 可赋予废弃区某位中士级赏金猎人——“裂隙行者”：表面上效忠于 PHARAOH-0 的令牌体系，实际上为铸刻者传递情报，用双面人硬币式的密文编码将每次行动信息夹在赏金报告里。代价：随时可能被 PHARAOH-0 的解析层看穿，届时会成为两边都不庇护的弃子。
