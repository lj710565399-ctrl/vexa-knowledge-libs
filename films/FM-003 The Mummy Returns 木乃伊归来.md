## 二、场景结构模式（可复用原型）

### 模式A：佩戴式倒计时装置
- **结构**：角色被迫佩戴（或植入）限时装置 → 倒计时驱动全片节奏 → 必须在时限内到达特定地点
- **关键**：装置本身是麦格芬也是计时器，物理绑定在角色身上无法摘除
- **VEXA适配**：直接对应 KAEL 的72小时倒计时——手腕导航器可以设计为类似手镯的不可拆卸装置，倒计时可视化（裂纹扩大/光芒减弱）

### 模式B：双线竞速叙事
- **结构**：正派和反派同时向同一目标移动，观众在两条线之间切换
- **效果**：天然的剪辑节奏驱动器，不需要额外的情节冲突就能维持张力
- **VEXA适配**：VEXA-09 从废弃区地表向第七层推进 vs PHARAOH-0 的清扫单位从深层向外扩散，双方在某一层相遇

### 模式C：沉睡军团召唤
- **触发**：特定仪式/装置激活 → 地面裂开 → 大量战士从地下涌出
- **视觉**：先听到地面隆隆声 → 裂缝 → 手臂伸出 → 军团成型 → 齐步前进
- **VEXA适配**：PHARAOH-0 激活废弃区的休眠机械单位，从墙壁/地板/天花板中成批涌出

### 模式D：前世记忆觉醒
- **结构**：角色通过闪回发现自己与古代历史的关联 → 获得新能力/新知识
- **VEXA适配**：VEXA-09 的装甲系统深层存有被封锁的记忆数据，在接触特定古埃及遗迹时触发解锁，揭示她与 PHARAOH-0 的历史关联

## 三、角色关系动力学

### 3.1 亲子绑定式紧迫感
- 孩子被绑架 + 身上绑着倒计时装置 = 双重焦虑叠加
- 父母的动力不需要任何额外解释，观众天然代入
- **VEXA适配**：虽然 VEXA-09 世界观不适合亲子线，但"同伴身上绑着倒计时 + 被敌人控制路线"的结构可直接套用（KAEL 被 PHARAOH-0 的信号劫持，被迫沿特定路线移动）

### 3.2 前世今生的能力继承
- 伊芙琳从"学者"升级为"战士"，合理性来自前世记忆
- 不是突然变强，是"一直有这个能力但被封印"
- **VEXA适配**：VEXA-09 的模块化装甲内可能存有被锁定的高级战斗模式，需要特定条件触发

### 3.3 反派内部的主从关系
- 伊莫顿与蝎子王是两个独立的威胁，伊莫顿的目标是杀蝎子王夺取军团控制权
- 三方博弈（主角vs伊莫顿vs蝎子王）比二元对抗更有戏
- **VEXA适配**：PHARAOH-0 内部可以设计分裂——核心意识 vs 失控的子系统，VEXA-09 利用内部矛盾破局
