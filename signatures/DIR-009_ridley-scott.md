# DIR-009 · 雷德利·斯科特 / 银翼杀手

## ① 一句话签名
烟雾里的霓虹与体积光、压迫的巨构都市、酸雨潮湿与东方招牌，反乌托邦赛博城市的氛围原典。

## ② 镜头语言颗粒
- **运镜**：缓慢庄严的推移/升降掠过巨构;烟雾中机位穿行；少手持，沉稳压迫
- **景别**：仰拍巨构都市的极大全景（人渺小于建筑）↔ 雨中霓虹下的面部特写；框中框（透过百叶/雨窗）
- **打光**：**体积光（god rays）穿烟雾**是签名——强光束穿过尘雾/雨幕、旋转探照灯扫过、霓虹招牌反射在湿地面；强逆光剪影
- **调色**：青黑暗调 + 霓虹暖点（橙红招牌 vs 冷蓝雾），高对比、脏质感、潮湿反光
- **节奏/剪辑**：缓慢、沉浸、靠氛围而非剪辑；长镜让都市奇观呼吸；电子氛围乐（Vangelis 式）
- **签名**：烟雾体积光、巨构都市、酸雨湿地面反光、东方招牌、旋转探照灯、反乌托邦压迫

## ③ VEXA-09 可复用改编
你的赛博埃及城市氛围原典：用烟雾体积光 + 巨构压迫拍废弃区第七层全景（古埃及式神殿巨构在尘雾中，旋转探照灯扫过，象形霓虹招牌反射在酸雨湿地面）→ 仰拍人渺小于巨构 → 青黑暗调 + 霓虹暖点。VEXA-09 雨中穿行、框中框透过百叶拍。最适合奠定整部片的赛博废土城市质感与反乌托邦压迫感。

## ④ 适用层 / 触发
**适用层**：赛博城市氛围 / 巨构定场 / 反乌托邦质感（喂 /shotlist 整场调性 + 定场）
**触发**：雷德利斯科特 / 银翼杀手 / 烟雾霓虹 / 酸雨 / 巨构 / 体积光
