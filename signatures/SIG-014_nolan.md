# SIG-014 · 诺兰 / 实拍时间

## ① 一句话签名
实拍质感、IMAX 巨幕、交错的时间结构与平行剪辑，宏大冷峻、智性张力拉满。

## ② 镜头语言颗粒
- **运镜**：稳定大景为主（IMAX 不晃），航拍/巨构定场 + 沉稳推移；手持仅用于战术紧张段；旋转/失重的实拍奇观
- **景别**：极致大全景（宏大实景奇观）↔ 紧张面部特写；强调真实空间的体量
- **打光**：自然冷峻光，硬朗写实，少柔光；实景光源
- **调色**：冷蓝灰、去暖、克制低饱和；质感扎实如纪录
- **节奏/剪辑**：**多线时间交叉剪辑**是签名——不同时间线/速度并行推进，节奏靠剪辑结构而非单镜；配乐（汉斯·季默式）持续低频推进制造焦虑；高潮多线收束
- **签名**：实拍奇观（不靠 CG）、非线性时间、平行蒙太奇、宏大冷峻、滴答倒计时焦虑

## ③ VEXA-09 可复用改编
用在 VEXA-09 宇宙的**宏大高潮 / 多线收束**：KAEL 的 72 小时生存倒计时天然适合诺兰式滴答焦虑——用交叉剪辑并行推进“倒计时归零 / VEXA-09 抢晶核 / PHARAOH-0 逼近”三线，冷蓝灰调、巨构大全景定场废弃区第七层、持续低频配乐推进。适合需要史诗体量与时间压迫感的全片高潮章节。

## ④ 适用层 / 触发
**适用层**：宏大高潮 / 多线收束 / 倒计时焦虑（喂 /shotlist 整场调性 + /dagang 结构）
