## 二、场景结构模式（可复用原型）

### 模式 1：科学意外觉醒 (Scientific Accident Awakening)

- **触发：** 主角因探索或误入禁区接触实验品（基因改造蜘蛛/生物类实验体）
- **结构：** 普通环境 → 意外接触 → 生理剧变 → 身体不适/混乱 → 发现自身变化后的测试流程
- **效果：** 构建主角的异化起点，同时用科学语言解释超能力产生逻辑
- **VEXA-09 适配：** VEXA-09 在废弃区第七层祭坛获得晶核能量器时，可触发核心程序 **类似觉醒效应**——能量器与 VEXA-09 的装甲芯片产生 **频率共振**，导致系统短暂过热/护目镜显示乱码（类似蜘蛛咬伤后的 "身体不适"），之后发现新功能解锁（如透视远古符文、快速扫描特定物质结构），对应彼得在浴室测试新能力的过程

### 模式 2：天台上层追逐 (Elevated Chase)

- **触发：** 反派/目标在都市上层建筑间快速移动，主角不得不在地标间展开垂直追逐
- **结构：** 发现目标 → 第一人称摆荡视角 → 障碍预测与躲避（脚手架/广告牌/通风管）→ 突然下坠悬停 → 威胁性对峙
- **效果：** 利用城市空间制造紧张感，凸显主角非人的敏捷与判断力
- **VEXA-09 适配：** 在 VEXA-09 与 PHARAOH-0 的第一次正面遭遇时，可设计在 **废弃区上层“霓虹街区”** 发生追逐。VEXA-09 利用跳跃与攀爬结构，PHARAOH-0 则在更高处用电线甩动和机械跳跃追击。镜头可用 **第一人称快速前进**（类似蜘蛛侠摆荡），以 Seedance 关键词：`第一人称视角, 城市屋顶飞跃, 快速回避障碍物`

### 模式 3：机械改造体对抗 (Cybernetic Adversary)

- **触发：** 反派使用机械外骨骼/强殖装甲增强战斗力
- **结构：** 首次接触被压制 → 观察其攻击模式（机械弱点）→ 利用环境限制其机动性 → 施加超负荷力量击破装甲
- **效果：** 蜘蛛侠利用敏捷避开蜥蜴人的重击，展现 "巧胜力" 的作战核心
- **VEXA-09 适配：** PHARAOH-0 本身是 **机械木乃伊守卫升级体**（更外露的铜导体电缆+古埃及雕塑风格装甲）。VEXA-09 可使用在祭坛中获得的能力：用 **高频脉冲** 短暂瘫痪 PHARAOH-0 的神经网络连接，然后用周边废墟金属结构（类似化学罐区）设陷阱使其过载。场景情绪参考蜘蛛侠学校决战：**被压制的绝望 → 找到规律 → 逆转胜**

### 模式 4：加速坠落救援 (Freefall Rescue)

- **触发：** 重要角色从高处坠落
- **结构：** 惊觉有人坠楼 → 瞬间反应并加速坠落 → 蛛丝/快速锁定目标 → 在触地前悬停（慢动作配合坠地前的一秒）→ 拉回+缓冲
- **效果：** 死亡倒计时压迫，利用慢动作展示救援动作的极限挑战与决心
- **VEXA-09 适配：** 在 **废弃区第七层祭坛战斗中**，KAEL 不慎从破损的地宫台阶坠落（类似格温坠楼），VEXA-09 需作出极限救援：在 **飞身坠落同时启动装甲推力**，拖住 KAEL 的背带，利用墙壁跳跃缓冲。此刻视觉应出现 **慢动作 + 碎裂装甲细节**，配以加速度数值 HUD 显示。参考镜头：蜘蛛侠在格温坠楼时的极限缓冲，但 VEXA-09 版更倾向于 **机械力学的沉重感**

## 三、角色关系动力学

### 关系组 1：彼得 + 格温
- **动态：** 天才组合 + 信任同盟。格温不是花瓶，她使用自己能力帮助彼得，并主动提供实验室入口和信息。两人关系从被吸引到互相依赖。
- **冲突点：** 格温父亲的警告（"她是我女儿"）+ 彼得对保护格温的执念（最终导致警长死亡）。
- **VEXA-09 适配：** 对应 **VEXA-09 + KAEL** 关系。KAEL 类似格温：是 **技术上能提供关键帮助的人类帮手**（格温提供实验室密码/KAEL 提供导航器频率扫描）。二人早期互相试探 + 中期依赖对方特长才能推进，最后 VEXA-09 倾向于主动保护 KAEL 的意识。冲突点应为：KAEL 是否愿意冒死帮助，以及 VEXA-09 的程序是否会因为在危机中优先保护 KAEL 而忽视任务目标。

### 关系组 2：彼得 + 康纳斯博士
- **动态：** 导师堕落成复仇者。最初彼得尊敬康纳斯（认同父亲的研究），但发现康纳斯已沉沦于力量，二人形成 "科学父辈阴影" 中的正反对抗。
- **冲突点：** 暴力科学 vs 伦理科学。康纳斯追求绝对力量消除弱点，而彼得用非致命战术与理念对抗。
- **VEXA-09 适配：** 转化为 **VEXA-09 + PHARAOH-0 (两者潜在同源)**。PHARAOH-0 是 VEXA-09 **改造技术的另一面**。PHARAOH-0 追求彻底控制与消除有机弱点（木乃伊守卫是无机机械化），而 VEXA-09 保持了对 "残存人性/不可控" 的容忍。结构上类似康纳斯与彼得：一个选择用非人化增强消灭人类缺陷，一个选择守护弱点。

### 关系组 3：彼得 + 警长史黛西
- **动态：** 法律与正义的不合作同盟。史黛西是警长，不喜欢私刑，但认可蜘蛛侠保护市民的动机。临终交接：父辈责任的传承与承诺。
- **冲突点：** 警长要求彼得远离格温，彼得无法遵守，最终导致警长为救他而死。
- **VEXA-09 适配：** 对应 **VEXA-09 + 铸刻者**（暂定）。铸刻者在 Caster's Bar 帮助二人，但也暗示 "我做这一切不会免费"。铸刻者可能像史黛西：用 **隐藏的规则** 约束 VEXA-09： "我帮你是因为你有用，但如果你越过我的底线，我也会是你的敌人。" 铸刻者也可能是最早认识 VEXA-09 制造者的人，承担父辈秘密传递者的角色。
