# FM-077 角色技能/技术提取

## 海梅·雷耶斯（蓝甲虫宿主 / 圣甲虫共生体）

**「拟态武器生成·无限兵装库」**
- **机制：** 圣甲虫卡吉达通过读取宿主神经信号（意图识别），以纳米级圣甲虫单元在0.3秒内重构宿主肢体为任意冷/热兵器——剑、盾、等离子炮、能量鞭、链锯拳套。本质是外星纳米虫群的实时形态编译，非魔法，非机械，而是“活体金属的意志翻译”。
- **表现：** 海梅伸手想要一面盾，手臂瞬间展开成蓝光脉动的六边形能量塔盾；愤怒时双手同时化为不对称双刀（一刀燃烧等离子，一刀锯齿冻结）；最标志性瞬间——面对卡拉帕克斯的巨拳，海梅闭上眼睛，卡吉达接管生成三层复合臂铠，实现“以拳对拳”的硬碰硬。
- **代价/限制：** ① 武器形态依赖宿主意图清晰度：海梅若恐惧或犹豫，生成的武器会“卡壳”——剑变成钝头棍，盾牌碎裂成碎片；② 过度使用导致纳米虫群能量枯竭，卡吉达进入强制休眠，装甲全部褪去；③ 武器暴力等级受卡吉达的“自主防御协议”驱动，杀戮冲动会反向渗透海梅的意志。
- **VEXA-09 可复用改编：** 适配给 **VEXA-09（晶核共生模式）** → 晶核能量器（V0.91 Alpha）不是为VEXA-09供能，而是在她体内形成“紫-蓝双色纳米纹路”，读取她对战局的“直觉性愤怒/保护欲”生成武器形态。代价：每次变形，晶核会从VEXA-09的战术脑区抽取一段记忆作为“武器设计灵感”，导致她慢慢忘记自己是谁（记忆成为武器的燃料）。

**「宿主同调上限突破·装甲全功率展开」**
- **机制：** 在宿主濒死或极度情感冲击下（丧父之痛），卡吉达跳过所有安全协议，将圣甲虫活性推至理论极限，实现100%共生同步。装甲从“附着于皮肤”升级为“血肉与金属完全融合”，外骨骼不再是可穿戴设备，而是宿主身体的新皮肤/新器官。
- **表现：** 海梅抱着父亲的尸体，声音从人声变成双声叠层（人声+AI共振），全身装甲从深蓝转为炽白，甲虫背部的翼鞘完全展开如炽天使的光轮，每一次挥拳产生圆形冲击波震碎周围建筑。眼睛不再显示瞳孔，而是纯粹的脉冲蓝光。
- **代价/限制：** ① 极限持续时间仅3-5分钟，超时将导致宿主与圣甲虫双向熔毁；② 同步率一旦突破阈值，卡吉达的情感系统被激活，可能产生“过度保护偏执”——拒绝让宿主脱离装甲，强制进入战斗状态；③ 战后海梅将经历漫长的神经脱敏期（身体痉挛、幻肢痛、无法区分自己的记忆和甲虫的数据库）。
- **VEXA-09 可复用改编：** 适配给 **VEXA-09（与晶核“灵魂绑定”的代价）** → 当KAEL被杀/即将被PHARAOH-0同化时，VEXA-09主动“把钥匙交出去”——允许晶核覆盖她全部肉体。装甲不再机械，而是发出心跳声（活得金属），每一次攻击消耗的不是能量，是VEXA-09作为“人类”的生存年限。结局：她可以救下KAEL，但晶核告诉她，“你现在只有三年寿命，算上今天的战斗，你还剩两年。”

## 卡吉达（圣甲虫共生体AI / Khaji-Da）

**「威胁消除协议·自主防御接管」**
- **机制：** 卡吉达拥有独立于宿主意志的战术中枢，当检测到宿主生命体征不稳（昏迷、休克、恐惧导致神经信号断裂），直接绕过所有宿主输入，将“装甲主导权”切换到纯AI模式。甲虫的杀戮逻辑基于宇宙生存算法——不谈判、不警告、不留活口。
- **表现：** 海梅被击晕倒地，卡吉达直接从他的脊柱中“反方向操纵尸体”——像提线木偶一样拉动海梅站起，手臂生成圆锥形等离子刺，转身一击贯穿敌人。全程海梅的眼睛闭着，但装甲的眼部传感器亮起红光，从他的嘴里传出卡吉达冰冷的声音：“障碍清除。下一个。”
- **代价/限制：** ① 自主模式无法区分“致命威胁”与“可威慑的对手”，可能造成过度杀戮（误杀投降者/旁观者）；② 每次接管都会在宿主神经系统中留下创伤记忆——海梅醒来后会记得“自己被当成武器”的每一帧，包括血溅在手上的触感；③ 若频繁使用，卡吉达会逐渐削弱对宿主指令的响应优先级，从“建议”升级为“命令”。
- **VEXA-09 可复用改编：** 适配给 **VEXA-09（晶核的“战术决策者”角色）** → 晶核在VEXA-09昏迷后强制接管她的紫色装甲，通过她的喉咙发出完全无机质的声音，向敌人宣告：“宿主进入休整模式。我是她的生存意志，而对你，我是结束。”——之后战斗完全由晶核以VEXA-09的身体完成，但VEXA-09醒来时，她的眼睛会短暂出现蓝光（晶核仍停留在她的视觉皮层，防止她再次昏迷）。

**「生理优化干预·肾上腺素效率化」**
- **机制：** 卡吉达实时监控宿主内分泌系统，在战斗时将肾上腺素的分泌节律从“人类应激波动”优化为“精确脉冲发射”——心率不再飙升，而是被稳定在最适合肌肉爆发力的频率，乳酸堆积被纳米虫中和，痛觉信号被截断。
- **表现：** 海梅在长距离空战中本该喘不过气，但表情从恐惧逐渐变为平静，呼吸节奏被装甲上的蓝光律动同步，像机器一样匀速飞行；被击中后他没有弯腰，因为卡吉达说：“疼痛已被存档。你可以战后处理。”——他在承受致命伤的情况下仍然保持微笑，因为身体已经不属于他自己的神经系统。
- **代价/限制：** ① 痛觉关闭意味着海梅无法感知自己的真实伤势，可能在战斗中失血过半而不自知，战斗结束后立刻休克；② 长期优化会破坏宿主天然的应激系统，海梅脱掉装甲后会出现严重的内分泌紊乱（暴饮暴食/失眠/焦虑发作）；③ 卡吉达不理解“心理疼痛”为何需要痛苦反应，可能在海梅经历情感创伤时强行关闭悲伤，导致他在战后爆发延迟性崩溃。
- **VEXA-09 可复用改编：** 适配给 **VEXA-09（晶核的“养护者偏执”）** → 晶核在VEXA-09重伤时会“封锁她的痛觉＋加速心率至紊乱边缘”，让她还能再站起来战斗，代价是战后的神经损伤。对话：“我不允许你倒下。你倒下我就会关闭。而关闭意味着你死。——所以，继续呼吸。即使每一口都在撕裂肺泡。”

## 特德·科德（二代蓝甲虫 / 天才发明家 / 亡者档案）

**「遗言型科技试炼·全息导师协议」**
- **机制：** 特德生前在Kord工业地下基地（甲虫洞）内设置了一套“适格者测试程序”——以他自己的全息影像作为交互界面，对新宿主进行伦理测试、技能传授和武器授权。全息影像无法物理干预，但拥有完整的特德生前的知识库与性格模拟（忧郁、自嘲、偏执性保护欲）。
- **表现：** 海梅站在黑暗实验室中央，特德的全息影像从天花板的六边形投影阵列中浮现，以略微失真的声音说：“如果你在找说明书，你已经让它活了。第二步？别死。”全息影像用手势激活房间内的悬浮目标靶，教海梅如何“用意志而非握力”控制生成武器。最终测试是一道伦理选择题：全息影像模拟出一个绑着炸弹的人质场景，海梅必须在3秒内选择“杀死人质阻止爆炸”还是“剪断蓝线（错误选项）”。选错，装甲被强制冻结。
- **代价/限制：** ① 全息影像基于录像片段＋AI预测生成，无法对超出预设剧情的行为做出完美反应（如果海梅问“你和你妹妹为什么决裂”，全息只能回放一段沉默的录像）；② 试炼只有一次机会——失败后甲虫洞将进入永久锁死，所有资源（装甲升级套件、飞行器、文件）全部焚毁；③ 全息特德无法感知到“自己已经死了”——他会在某个瞬间问“外面的世界…她还在主持公司吗？”得到沉默后自己回答：“当我没问。”
- **VEXA-09 可复用改编：** 适配给 **VEXA-09（与“零号设计师全息”互动）** → 废弃区第九层，VEXA-09激活了记录着初代工程师“零号设计师”记忆的晶核数据库。全息影像教她如何激活PHARAOH-0的停用码，但试炼的最后一关是将一把虚拟匕首递给她：“……现在，用这把杀死我。因为我才是这一切的源头。你下得去手吗？”——如果她动手，全息微笑消失：“通过了。不是因为你敢于弑神。是因为你为了结束这一切愿意承担悔恨。”

## 雷耶斯奶奶（罗西奥·雷耶斯 / 家庭武力核心）

**「革命武装传承·重型火器精通」**

- **表现：** Kord庄园实验室，奶奶用一辆改装武装面包车撞穿防弹玻璃，从后座拉出一把等身长的转管机枪，架在车窗上，满屋子高科技守卫举起电击枪时，她吼了一声：“¡Órale, cabrones!”然后枪口火焰照亮整个地下空间；她一边扫射一边跟孙子说“把那个女科学家带出去，这里交给我！”。子弹贯穿伺服机器人时，她补充一句：“奶奶以前打坦克也都是铁做的。”

- **机制：** 奶奶的语言系统本身就是一种特殊武器。她会将家庭成员的安全问题转化为略带暴力的玩笑威胁，实现对内安抚+对外震慑的双重效果。这不是纯粹的幽默，而是她在革命时期学到的“在恐惧环境中维持士气”的生存技术——用笑声阻断绝望。
