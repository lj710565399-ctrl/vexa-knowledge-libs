## 二、场景结构模式（可复用原型）

### 模式A：共生体侵袭→自我镜像
- **触发**：外来黑暗力量（共生体）附着主角，放大其负面欲望
- **结构**：傲慢 → 失控 → 伤害身边人 → 觉醒 → 自我剥离
- **效果**：主角同时是受害者与加害者，战斗既是外部打斗也是内心挣扎
- **VEXA适配**：VEXA-09 在废弃区深层接触到PHARAOH-0残留精神碎片，处理器被"共感覆盖" — 瞬间获得巨大力量但开始对KAEL产生敌意，需通过自主重启/KAEL唤醒剥离感染

### 模式B：三角复仇链
- **触发**：反派A（哈里·奥斯本）因家族血仇追杀主角，反派B（弗林特·马克）因女儿生存动机与主角对抗，反派C（埃迪·布洛克）因职业嫉妒及共生体裹挟
- **结构**：三线同时推进，每条反派剧情独立但"终局合并"
- **效果**：主角没有闲余时间，每次解决一个反派等于释放另一个
- **VEXA适配**：废弃区第七层同时面临三股势力 — PHARAOH-0清扫单位（主敌）、暗网赏金猎人团（因VEXA-09叛逃而被悬赏）、地下铸刻者联盟（看似盟友但立场暧昧），三线交错时间窗

### 模式C：身份互换伪装
- **触发**：彼得·帕克用蛛丝换装技巧冒充"自我"来获取信息
- **结构**：伪装 → 误导（包括MJ） → 暴露 → 信任重建
- **效果**：主角与盟友之间产生"认不出"的致命空隙
- **VEXA适配**：VEXA-09 废弃区中遇到伪装成PHARAOH-0特工的"盟友"，或反之 — KAEL 被迫使用晶核能量器伪装成诺克底斯之眼成员潜入地下城堡，但能量器反噬暴露身份

### 模式D：重力决战—高空坠落
- **触发**：哈利与彼得从70层高的大厦坠下，空中抱在一起打斗
- **结构**：垂直空间作为战场 → 重力视为敌人（双人一起坠） → 物理解法（蛛丝抓梁柱）→ 合作接住对方
- **效果**：将盟友/敌人的物理连接转化为情感连接
- **VEXA适配**：废弃区最深层直通深渊坠落 — 晶核能量器重力稳定装置损坏，VEXA-09 和 KAEL 一起坠落，必须在高速自由落体中建立连接（护甲缠绕/携手启用回收推进器）

## 三、角色关系动力学

### 3.1 共生体纽带（毒液与宿主）
- **载体**：黑色液态附着、声音变低沉、白色眼睛和牙齿狰狞
- **功能**：共生体即是反派也是主角的黑暗半身—被打倒的不是敌人，是自己的阴暗面
- **三次递进**：① 附着前 → 彼得情绪波动共生产生回响 ② 附着中 → 力量自大但无法自控 ③ 剥离 → 需要外力与内在觉悟共同完成
- **VEXA适配**：PHARAOH-0 操作系统可以使用类似共生体机制 — VEXA-09 的装甲核心处理器被反向写入，核心代码被PHARAOH-0子程序覆盖，KAEL必须在导航器中找到"剥离指令"并远程注入，同时VEXA-09必须主动压制自己的攻击冲动

### 3.2 复仇三角 → 和解菱形
- **起点**：哈利（丧父仇）vs 彼得（失兄怨）vs 弗林特（本叔之死偶然性未明）
- **转化**：哈利 → 记忆恢复选友情 → 被迫与新妖精身份冲突；彼得 → 发现真相后宽恕弗林特
- **终点**：仇恨并不消失，只是被选择放下
- **VEXA适配**：VEXA-09 和 KAEL 的关系定位 — VEXA-09 最初是因胁迫/利益组队，但地宫事件后两人深度绑定，VEXA-09 潜意识中对"被遗弃"的恐惧（来自制造者）与 KAEL 对"晶体续命"的焦虑共同构成微型复仇弧

### 3.3 自我镜像（MJ vs 格温·斯黛西）
- **载体**：两位女性角色对彼得的态度差异 → 纯洁崇拜 vs 深切了解
- **功能**：格温是彼得的理想化自我投射，MJ是真实自我
- **VEXA适配**：在废弃区，KAEL 可能遇到"理想版VEXA-09"（由PHARAOH-0虚构的全盛赏金猎人投影）对比真实VEXA-09（破损、脆弱但真实），KAEL必须选择相信真实而非完美
