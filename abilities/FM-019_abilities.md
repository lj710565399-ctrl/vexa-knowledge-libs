# FM-019 角色技能/技术提取

## 莱克茜·伍兹（高海拔生存专家 / 探险向导 / 最终幸存者）
**「危机词汇解构」**
- **机制：** 战技——将致命威胁即时翻译为冷血的专业术语标签（kill box、pressure cooker、predator-prey dynamic），以此在认知层面剥离恐惧的情绪附着，使自己在极端恐慌中仍能进行战术性逻辑推演。
- **表现：** 当团队陷入混乱时，莱克茜用一句短促的地理/生物学术语重新框架化整个局势，周围人的非理性尖叫与她的冷声定义形成鲜明反差。标志性瞬间：在异形猎杀开始时，她对着慌乱开枪的队员说出“We’re in its kill box”，所有人瞬间理解了自己在空间中的猎物位置。
- **代价/限制：** 对恐惧的过度认知隔离使其在危机后难以回归正常情感反应，呈现代谢性情感钝化；队友可能因她的“冷血”而误判她为缺乏共情者，导致团队信任裂痕。术语库仅覆盖已训练场景，遭遇完全未知的超逻辑威胁（如铁血战士的仪式性行为）时会短暂失语。
- **VEXA-09 可复用改编：** 可赋予**铸刻者**——当地宫机制触发时，铸刻者用冷硬的考古学术语命名暗物质现象（“这是三相能量泄流”“现在我们处于献祭回廊的第三象限”），将恐惧转化为分类学操作。代价是他对KAEL的情感需求完全误读，以为“解释清楚即安全”。

**「极地职业冷静」**
- **机制：** 类战技——在高海拔/极端环境训练中形成的生理控制模板，自动压制惊叫、喘气、无效动作等消耗性反应，将身体调节至低氧耗、低暴露的生存模式。
- **表现：** 异形破胸体从队友胸腔钻出时，莱克茜脸部肌肉仅收缩0.3秒便恢复平稳，呼吸频率控制在与高海拔匹配的慢节奏，而非惊恐过呼吸。动作精准不浪费——关闭阀门、计算通路、后退三步、转身奔跑，每一步都在氧耗预算内。
- **代价/限制：** 生理压制过度时，他人无法从她身上读取任何早期危险信号——她看起来永远“能搞定”，直到临界崩溃点才突然表现出全面衰竭。冷静模板是训练产物，会因某些触发条件（队友尖叫特定频率、血腥味浓度超标）而发生瞬间失压，导致压制后的情绪反噬更猛烈。
- **VEXA-09 可复用改编：** 可赋予**KAEL**——KAEL的导航器内置“环境压力自适应算法”，当他进入地宫深层时自动缩减非必要生理输出（体温降低、脉冲减缓），使他看起来像一个冷静的数据终端。代价是当暗物质能量场干扰该算法时，KAEL会像被拔掉电源一样骤然陷入超载式恐慌，无法像VEXA-09那样纯粹靠本能战斗。

## 刀疤 / Scar（年轻铁血战士试炼者）
**「狩猎仪式嵌套」**
- **机制：** 战技 + 装备辅助——遵循铁血狩猎法典，将整场猎杀拆解为“追踪→标记→荣誉击杀→战利品采集→仪式认证”五个环环相扣的仪式步骤。每一步的完成质量直接影响最终评分（由自身装甲记录仪和飞船长老远程评估），而非单纯追求击杀数。允许在猎物未构成威胁时保持隐身不杀（非猎物不杀原则），也允许在特定条件下将非铁血个体标记为“荣誉同盟”。
- **表现：** 刀疤在猎杀前会用面具扫描目标，确认其是否符合“适格猎物”标准（会优先躲避抱脸虫而非主动攻击的个体被剔除）。击杀异形后用其酸血在自己面具上刻下标记，或从尸体上割下特定外骨骼部分收入战利品袋。最标志性表现：用面具投射光纹在莱克茜前额点亮铁血纹章，完成“非猎物标记”，周围铁血同族看到此标记后不再将其视为目标。
- **代价/限制：** 严格遵守仪式意味着在极端危机下可能失去战术灵活性——猎物逃入“非狩猎区”（如人类密集的狭窄通道）时不能追击；自己受伤后必须优先清除血迹防止异形追踪，而非继续攻击。与同族共享猎物荣誉规范，若他人抢杀自己标记的猎物可能触发内部决斗。仪式完成度不足会导致返回后被长老认定试炼失败，面临降级甚至流放。
- **VEXA-09 可复用改编：** 可赋予**PHARAOH-0**——PHARAOH-0遵循古埃及“审判者代码”，“适格威胁”和“非目标”之间有严格划分。当它扫描VEXA-09并识别出其装甲核心与自身同源时，立即停止攻击，并在其护目镜上投射一个类似荷鲁斯之眼的纹章，将其标记为“碎片/继承者”。代价是代码不可被外部修改，若铸刻者或KAEL试图攻击被标记后的VEXA-09，PHARAOH-0会瞬间判定他们为“亵渎者”并切换为灭杀模式。

**「肩炮自适应校准」**
- **机制：** 装备——肩部等离子炮通过面具视觉追踪系统锁定目标，自动计算移动轨迹、环境障碍、酸血溅射范围后完成一击必杀。非战斗状态可折叠收入肩甲，展开时有独特嗡鸣声作为预热反馈。三种发射模式：单发精准击杀、三连发压制、持续照射烧蚀。
- **表现：** 黑暗中先出现三个红色激光三角点落在目标头部，随后嗡鸣升频、蓝白等离子团沿激光轨迹击中目标。标志性瞬间：异形女皇追击莱克茜时，刀疤转身、肩炮展开、红光点落在皇后头部，但他选择击中天花板制造冰层塌陷而非直接击杀（弹量不足时的非致命环境利用）。
- **代价/限制：** 每次发射后有蓄能间隔（约2-3秒），期间无法连续攻击；持续照射模式会快速耗尽能量模块。酸血溅射对肩炮结构有腐蚀风险，击杀异形后需要立即做清洁维护。等离子团的光迹在黑暗中会暴露自身位置，与隐身战术矛盾。
- **VEXA-09 可复用改编：** 可赋予**VEXA-09**——将肩炮转化为VEXA-09的“右臂晶核投射器”，激活时从装甲前臂展开一个短程能量炮口，投射出紫色晶核能量弹。蓄能间隔对应为VEXA-09核心处理器的冷却周期，若连发会导致义体过热反馈回她的神经接口。每次发射的光迹会点亮地宫墙壁上的象形文字刻痕，意外触发隐藏机关。

**「终末净化协议」**
- **机制：** 义体/装备——铁血战士装甲内置的自毁装置，激活后产生范围性湮灭脉冲（蓝色光球扩散），可清除周围一切生命体。设计初衷是在被寄生或捕获时以光荣方式终结，避免自身尸体被敌人利用。
- **表现：** 失败后的铁血战士单膝跪地，按住腕部装置启动倒计时——装甲缝隙间开始泄出蓝白色光雾，脉冲扩散成半球形波纹，半径内异形卵因高温蒸发、酸血被冲击波吹飞。刀疤启动自爆时，面罩内部显示铁血语倒计时符文，长老在飞船中沉默注视其信号。
- **代价/限制：** ⚠️不可逆——一旦启动无法中止；启动者必然死亡。脉冲范围固定，若目标在范围外则无伤。长老评估失败试炼者时可能视此为“逃避后续考核”而非荣耀，引发铁血社会内部的荣誉争议。
- **VEXA-09 可复用改编：** 可赋予**VEXA-09**的核心装甲过载协议——当地宫第七层无法以战斗突破、晶核能量器过度激活导致暗物质泄露时，VEXA-09可选择“核心熔毁”，以自身处理器为引爆点释放一道清理脉冲，蒸发范围内所有木乃伊守卫和暗物质寄生物。代价是她的核心将进入数小时强制休眠期（非死亡，但等于将自己置于完全无助状态），且每次执行后她的“人性残留”指标下降一档。

## 查尔斯·毕肖普·维兰德（维兰德公司创始人 / 探险赞助人）
**「资本冻结协议」**
- **机制：** 装备+口语权术——以财务投资和雇佣契约为框架，在危急时刻将人命的存续权转化为可谈判的商业条款。利用“探险是受资助的”“你是我雇佣的”等话术，将逃亡重新定义为违约，将服从重新定义为履约。
- **表现：** 当莱克茜建议撤离时，维兰德平静地提醒“This expedition is financed. It is not optional.”——他用极低的音量和不可商榷的陈述句，把雇佣兵原本的逃生本能压回了服从状态。另一个标志瞬间：在被异形重伤后，他对莱克茜说“You were right. I should have listened”，这是他唯一一次让渡绝对权威，同时这本身也是一次谈判——用认错换取临终情报的获取。
- **代价/限制：** 资本逻辑只在文明社会框架内有效，对铁血战士（听不懂商业术语）和异形（不谈判）完全无效。团队中存在反叛阈值——当死亡概率超过工资的心理对价时，雇佣兵会选择撕毁隐性契约。他的权威最终依赖身边是否有武力执行者，一旦失去武装护卫，商业话术立即失效。
- **VEXA-09 可复用改编：** 可赋予**铸刻者**——铸刻者作为地宫考古行动的资金方和技术支持方，在危机中用“这才是计划的核心阶段”“你签的契约不是考古，是见证”将KAEL和VEXA-09的行为约束在既定模板内。他对KAEL的情感关怀是真实但次级的：“我原本想保护你”这句晚来的坦白在叙事上高度接近维兰德临终道歉。代价是当契约对象（VEXA-09）的自主意识突破阈值时，铸刻者的权威框架将瞬间崩塌。

## 异形皇后 + 异形族群（寄生性猎杀者 / 终极威胁）
**「活性寄生链投送」**
- **机制：** 生体武器——异形皇后产下的卵感应宿主靠近时，内部抱脸虫以爆发性弹射方式击发，精准落在宿主面部。抱脸虫植入寄生体后宿主暂时恢复正常意识，寄生体在胸腔内吸收宿主DNA完成形态定制，最终撕裂宿主胸腔钻出（破胸体）。破胸体在数小时内迅速蜕变为成年异形，具备猎杀能力并继续扩大寄生链。至此一个宿主变为三个猎杀单位（抱脸虫+破胸体+成年异形）。
- **表现：** 卵膜撕裂声 → 湿黏触手弹射音 → 宿主倒地喉间窒息声 → 短暂苏醒后的虚假安全 → 胸腔从内部鼓起的慢推镜头 → 肋骨折断 + 血溅 + 双颚首次弹出啸叫。繁衍效率极高，在封闭空间内一小时可将十人团队转化为一支异形小队。
- **代价/限制：** 寄生过程依赖宿主生物活性——死人不能被寄生（除非刚死），重度受伤的宿主会导致破胸体孱弱或死亡。寄生链稳定后，异形成体需要皇后持续释放信息素维持群体秩序，脱离信息素覆盖范围的异形会退化为无序攻击。异形卵对携带铁血装甲热能信号的个体有识别延迟（铁血体温低），初期容易被误判为“非宿主”。
- **VEXA-09 可复用改编：** 可赋予**PHARAOH-0的木乃伊守卫群**——木乃伊守卫的铜质电路绷带下封存着“暗物质寄生孢子”，当守卫被击杀且绷带断裂时，孢子以电弧形式释放并寻找最近的有机体（KAEL）或半有机体（VEXA-09的神经接口）作为新宿主。寄生后不在胸腔孵化，而是在宿主皮肤下生成古埃及象形文字状的暗纹，最终从眼窝或手指缝隙钻出微型光子幼虫。代价是这一寄生链需要晶核能量器持续供能，能量器离开特定范围时所有寄生单位会停滞。

**「酸血环境改写」**
- **机制：** 生体能力——异形血液为高腐蚀性分子酸，当躯体受到切割或枪击伤害时，酸血以高压喷溅或滴漏形式飞出，可烧穿多层金属/岩石/冰层。这一特性使“杀死异形”本身就成为一个环境破坏事件，杀伤异形的人必须即时规避溅射，否则同步自毁。
- **表现：** 异形被铁血腕刃切断肢体时，绿色酸血弧形喷出，滴落在地板上升起白烟并烧蚀出贯穿性孔洞。酸血滴穿金字塔冰层形成垂直通道——敌人被杀，但战场被改造。标志性零重力场景：飘浮的异形被击中后，酸血以球状微重力形态四散，击中墙壁后继续烧蚀，在金字塔内部打出一个一个随机出口。
- **代价/限制：** 酸血无差别伤害——异形自己不受自身酸血影响（表皮具有抗酸性），但同族异形可以相互灼伤。酸血暴露空气后活性随时间衰减，数分钟后腐蚀性降至不足初始的20%。封闭空间内击杀多只异形会导致大量酸血汇聚，引发结构性坍塌，猎杀者可能被自己的杀戮结果压死。
- **VEXA-09 可复用改编：** 可赋予**VEXA-09的晶核能量器**——当VEXA-09过载攻击木乃伊守卫时，守卫断裂电路喷出的不是酸血，而是高导性纳米液体，腐蚀装甲板和地面，将战斗区域逐层烧穿。优势：创造垂直逃生/进攻通道。代价：烧穿的楼层可能释放下方更危险的敌人，或导致地宫结构不稳定引发连锁坍塌。

**「完美生物静默猎杀」**
- **机制：** 生体能力——异形成体的外骨骼完全不反射常规探测声波和红外信号（冷血+吸音表面），可以在黑暗中完全隐形于热成像和声呐扫描。配合分泌的少量黏液消除脚步振动，进入猎杀状态时的异形不发出任何可被人类听觉捕捉的声音。
- **表现：** 人类扫描仪显示通道空无一物，但异形正倒悬在天花板上，双颚缓慢张开，酸性唾液滴落在受害者肩膀上——受害者感觉到液体时才抬头，已经在攻击范围正中央。标志性瞬间：在金字塔的曲折通道中，异形从黑暗角落无声滑出，尾部在墙上留下一道浅痕却不产生任何刮擦声。
- **代价/限制：** 静默状态需保持低速移动，快速奔跑时会暴露刮擦声和呼吸声。分泌黏液覆盖全身有干燥时间限制，长途静默猎杀后需返回潮湿环境补充体液。少数精密装备（铁血面具的多光谱视觉、赛博世界观中的电磁场感应器）可突破这一隐形。极度依赖阴影和垂直攀附面，在完全空旷明亮空间中静默猎杀失效。
- **VEXA-09 可复用改编：** 可用于地下遗迹中的**暗物质寄生体**——这些光子幼虫在孵化后可在墙壁的象形文字凹槽中完全无声移动，仅在被晶核能量器的蓝光照到一瞬间投射出拉长的蜘蛛形阴影。KAEL的导航器扫描显示“无生物迹象”，但他的肉眼看到有东西在文字中滑动，形成认知分裂恐惧。在VEXA-09的装甲感应器被暗物质干扰波段覆盖的区域内，她也会短暂失去对寄生体的追踪能力。
